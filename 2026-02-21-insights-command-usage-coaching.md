# Claude Code `/insights` — AI 사용 습관을 코칭하는 명령어

## 출처

- 자막: 한국어 자동 자막 (타임스탬프 포함)

## 한 줄 요약
Claude Code의 `/insights` 명령어는 지난 한 달간의 대화 기록을 분석해 사용 습관·마찰 지점·만족도를 리포트로 만들고, 바로 붙여 넣을 수 있는 맞춤 설정 규칙까지 제안하는 자기 성찰형 코칭 기능이다.

## 메타
- 영상 유형: 혼합형 (기능 개념 설명 + 사용법 안내 + 한계 리뷰)
- 핵심 키워드: `/insights`, 마찰 분석, 만족도 평가, CLAUDE.md 규칙 자동 제안, 할루시네이션 주의

## 핵심 개념

### `/insights`가 하는 일
화자는 이 기능을 PT(퍼스널 트레이닝)에 비유한다.

> "여러분 PT 받아보신 적 있으세요? 운동 잘하는 것 같은데도 이 동작이 문제예요, 이렇게 바꾸세요 하고 콕 집어 주잖아요. 근데 이게 AI 코딩 도구에서 실제로 일어나고 있어요." [00:00]

> "클로드 코드에서 슬래시 인사이트라고 치면 실행되는 분석 기능이 있거든요." [00:21]

(자막 "슬래시 인사이트"는 `/insights` 명령어를 가리킴 — 영문 표기 보정. 이하 동일.)

기능 정의:

> "여러분이 한 달 동안 AI한테 어떤 질문을 했는지, 어디서 막혔는지, 대화가 어떤 식으로 흘렀는지를 AI가 쭉 훑어보고 읽기 좋은 리포트로 만들어 줍니다." [00:26]

화자는 단순 사용량 집계와 선을 긋는다.

> "이건 단순히 이번 달에 AI를 몇 번 썼어요, 돈 얼마 나와요 이런 게 아니에요. 당신도 자꾸 막히더라, 이 부분은 이렇게 접근하면 훨씬 부드럽게 돼요라고 개선 방법까지 제안해 줍니다." [00:36]

> [의견] "AI 코딩 도구들 중에서 이런 자기 성찰 기능을 넣은 건 클로드 코드가 처음이에요." [00:46]

### 내부 작동 — 6단계
화자의 비유: "일기장을 읽어서 분석 리포트를 써 주는 비서" [00:56].

1. **대화 기록을 모은다** — 컴퓨터에 쌓인 AI 대화 기록을 불러오되, "너무 짧거나 잠깐 나눈 건 제외하고 의미 있는 것들만" 골라온다 [01:00].
2. **기본 정보를 뽑는다** — 대화 길이, 작업 종류, 사용 빈도를 정리 [01:09].
3. **대화가 길면 요점만 줄인다** — "두꺼운 책을 요약본으로 만드는 것처럼" [01:16].
4. **만족도 채점 (핵심)** — AI가 각 대화를 읽고 "이 사람이 대화에서 만족했나, 어디서 답답해했나, 결국 원하는 걸 얻었나를 평가" [01:20].
5. **패턴 종합** — 한 달치 대화를 전부 종합해 패턴을 찾아낸다 [01:28].
6. **리포트 출력** — "예쁜 리포트로 만들어서 브라우저로 띄워" 준다 [01:32].

## 리포트 구성 (카탈로그)

화자가 실제 보고서를 열었을 때 나오는 항목들 [01:39 ~]:

| 항목 | 내용 |
|---|---|
| **한눈에 보는 요약** | 잘 되고 있는 것 / 막히는 지점 / 바로 적용 가능한 개선 방법을 핵심만 [01:40] |
| **통계 대시보드** | 주고받은 메시지 수, 바꾼 코드 줄 수, 건드린 파일 수 등 숫자 [01:48] |
| **마찰(friction) 분석** | "진짜 흥미로운 건 마찰 분석" — 실제 사용자 보고서 기준 [01:56] |
| **만족도 평가** | "짜증난"부터 "행복함"까지 다섯 단계로 세션별 만족도를 AI가 판단 [02:15] |
| **붙여 넣기 가능한 설정 규칙** | "제일 실용적인 게" — 보고서 마지막에 추가하면 마찰이 줄어들 규칙을 구체적으로 제안 [02:22] |

### 마찰 분석 — 1순위는 "성급한 행동"
> "실제로 써본 사람들의 보고서를 보면요, 마찰의 1순위가 뭔지 아세요? 성급한 행동이에요. 46건이나 이거였다고 해요." [01:59]

> "쉽게 말하면 AI가 내 말을 다 듣지도 않고 바로 뭔가를 하려 덤벼드는 거예요. ... 야 잠깐만, 내가 원한 건 이게 아닌데 하는 그 순간이요." [02:06]

(46건 수치는 화자가 인용한 특정 사용자 보고서의 사례이며, 일반화된 통계가 아님.)

## 한계·주의점

화자가 "솔직한 리뷰"임을 강조하며 짚는 문제들.

### 1. 숫자를 지어낼 수 있다 (할루시네이션)
> "가장 큰 문제는 뭐냐면 숫자를 지어낼 수 있어요." [02:38]

해외 커뮤니티 사례:

> "자기가 실제로 클로드 코드를 10번 정도밖에 안 썼는데 보고서에는 300번 이상의 데이터베이스 작업 세션이라고 나온 거예요. AI한테 직접 이 숫자 맞아라고 물어보니까 이렇게 답했대요. 336이라는 숫자는 제가 만들어 낸 겁니다." [02:43]

원인:

> "이 분석에 비용을 줄이려고 작은 AI 모델을 쓰거든요. 근데 데이터가 부족하면 그럴듯한 숫자를 지어내는 현상이 생기는 거예요. AI의 고질적인 문제인 할루시네이션이요." [02:56]

화자 권고:

> "그래서 보고서의 구체적인 숫자보다는 패턴이나 경향성을 중심으로 보시는 게 맞아요." [03:06]

(자막 "작은 AI 모델" — 외부 보강 참조: 실제로는 Haiku 모델을 분석에 사용한다고 알려져 있음.)

### 2. 버그·최근 편향
> "또 하나 버그도 좀 많고요. 최근에 사용한 것들 위주로 분석이 쏠리는 경향도 있다는 피드백이 있습니다." [03:10]

## 왜 그래도 쓸 가치가 있나

> "한 블로거가 이런 평가를 했어요. 잘 훈련된 매니저에게 피드백 받는 것 같다. 이게 진짜 핵심이에요." [03:18]

> [의견] "지금까지는 AI 얼마나 썼는지, 돈 얼마나 드는지 다른 도구로 볼 수 있었어요. 하지만 너 이런 식으로 쓰면 더 효율적이야라고 코칭해 주는 건 이 기능밖에 없거든요." [03:25]

> [의견] "특히 맞춤 설정 규칙을 자동으로 만들어 주는 게 진짜 실용적이에요. AI 쓰는 방식을 최적화하고 싶어도 어디서 어떻게 건드려야 할지 모르잖아요." [03:33]

## 환경 셋업 / 사용 조건

사용법은 단순하다.

> "클로드 코드 켜시고요, 슬래시 인사이트 치시면 끝이에요." [03:44]

단, 조건이 있다 [03:49 ~]:

1. **Claude Pro Plus 또는 Max 구독**이 있어야 한다.
   (자막 "클로드 프로 플러스나 맥스 구독" — 보정. 정확한 플랜 명칭은 외부 보강 참조.)
2. **최소 2~4주** 정도 Claude Code를 써서 데이터가 쌓여야 한다. "처음 쓰자마자 돌리면 볼 게 없거나 해요."
3. **최신 버전**이어야 한다. 안 되면 업데이트.

결과는 컴퓨터 어딘가에 저장되고 자동으로 브라우저에서 열린다 [04:03].

## 화자의 총평

> [의견] "인사이트는 클로드 코드만의 독특한 기능이에요. 다른 AI 코딩 도구에는 아직 없습니다." [04:08]

> [의견] "다만 숫자를 맹신하지 마세요, 패턴과 [불명확: 원문 "제한", '제안' 또는 'trend'를 의도한 것으로 보이나 불확실] 위주로 보시고요. 맞춤 설정 규칙 [불명확: 원문 "제한", '제안'으로 추정되나 불확실]은 진짜 실용적이니까 적극적으로 활용해 보세요." [04:18]

> [의견] "아직 초기라 완벽하진 않지만 방향 자체는 저는 정말 좋다고 보고 있어요. AI를 더 잘 쓰기 위해 AI를 쓴다, 이 컨셉 자체가 앞으로 AI 도구의 표준이 될 수도 있거든요." [04:25]

## 외부 보강

영상의 자막 보정(작은 모델, 플랜 명칭, 명령어 철자)을 공식·커뮤니티 자료로 교차 확인했다. 영상 내용과 분리해 둔다.

- `/insights`는 로컬 세션 기록(약 30일치)을 읽어 인터랙티브 HTML 리포트를 생성하는 Claude Code 내장 명령어다. 외부 추적 없이 머신에 이미 저장된 데이터만 사용한다.
- 분석 파이프라인: 메타데이터 추출(토큰·도구·소요시간·수정량) → LLM(**Haiku**)을 통한 정성적 transcript 분석 → 세션 간 데이터 집계 → 멀티 프롬프트 분석. 영상이 "작은 AI 모델"이라 한 것은 이 Haiku를 가리킨다.
- 마찰 분석은 사용자 자신의 발언을 인용하며 반복되는 실수 패턴을 짚어 준다. 추천 항목으로 **여러 세션에 걸쳐 반복된 지시를 기반으로 한 맞춤 CLAUDE.md 규칙**을 생성해 붙여 넣을 수 있게 제공한다 — 영상이 강조한 "붙여 넣기 가능한 설정 규칙"과 일치.
