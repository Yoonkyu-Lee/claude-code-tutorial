## 핵심 개념

### Cowork = "답을 주는 도구"에서 "일하는 직원"으로
화자는 기존 Claude와 Cowork를 백과사전 대 신입 직원으로 대비한다.

> "우리가 지금까지 쓰던 Claude는요. 뭘 물어보면 답을 주는 도구였어요. ... 아무리 잘해도 결국엔 답변을 받는 거였어요. 근데 Cowork는 답변을 주는 도구가 아니에요. 나를 대신해서 일하는 직원을 만드는 도구예요." `[01:02]`

> "비유하자면요. 지금까지 Claude는 백과사전이었어요. 펴보면 답이 나와요. 근데 Cowork는 신입 직원이에요. 시키면 일을 해요. ... 백과사전은 내가 직접 일을 해야 되잖아요. 근데 직원은 내가 안 해도 일이 돼 있어요." `[01:16]`

### 작업 폴더 = "거실 열쇠만 준 집"
Cowork 시작 시 Claude가 일할 폴더를 지정하는데, 화자는 보안 우려를 집-방 비유로 설명한다.

> "내 컴퓨터가 방이 여러 개 있는 집이라고 쳐요. 내가 Claude한테 열쇠 하나를 줬는데 그게 거실 열쇠예요. 그러면 Claude는 거실만 들어갈 수 있어요. 안방도 못 들어가고 서재도 못 들어가요." `[02:22]`

> "Claude가 내 컴퓨터를 통째로 본다가 아니라 내가 허락한 폴더만 본다인 거예요." `[02:38]`

### MCP = "USB-C 타입"
연결 규격으로서의 MCP(Model Context Protocol)를 충전 케이블 통일에 비유한다.

> "옛날에 이 충전선이 다 달랐어요. 아이폰은 8핀, 갤럭시는 5핀. ... 근데 요즘은 어때요? USB C 타입 하나면 다 돼요. ... AI 세계도 똑같았어요. AI랑 다른 서비스를 이어주는 규격이 다 제각각이었거든요. [근]데 최근에는 MCP라고 해 가지고 모델 컨텍스트 프로토콜이라는 친구가 나왔는데 이 친구를 이용해 가지고 연결을 해 주면 됩니다." `[03:37]`

> "비유하자면요. 모든 콘센트가 USB-C로 통일된 거랑 똑같습니다." `[04:03]`

### 카카오톡 자동 전송 = "정문이 잠겨서 옆문(macOS 자동화)으로"
카카오톡은 텔레그램과 달리 개인용 API가 닫혀 있어, GUI를 사람처럼 조작하는 방식을 쓴다. **이 영상의 핵심 차별 트릭이다.**

> "텔레그램은 누구나 봇을 만들어서 자기 메신저랑 연결할 수 있는 API가 열려 있어요. 그래서 자동화하기가 쉬워요. 근데 카카오톡은 개인용 API가 좀 닫혀 있습니다. 회사가 정책적으로 그렇게 막아둔 거 같습니다." `[04:57]`

> "사람이 카톡 쓰는 방식을 그대로 따라하면 돼요. 앱 열고 친구 찾고 채팅창 열고 글자 치고 엔터. 이걸 Claude가 마우스랑 키보드로 직접 해 주는 거예요." `[05:03]`

> "비유하자면요. 카카오톡에 자동 메시지 보내는 일이 마치 회사 보안문 뚫는 거랑 비슷해요. 정문은 잠겨 있어요. 근데 옆문이 열려 있어요. 그 옆문이 macOS 자동화예요. 좀 돌아가지만 확실하게 들어갈 수 있습니다." `[05:18]`

### Cowork의 진짜 가치 = 단일 작업이 아니라 워크플로
> "[의견] 기존에 Cowork 하면 폴더 정리, 엑셀 만들기 이런 단일 작업 위주로 많이 보여줬잖아요. 물론 그것도 굉장히 좋습니다. 근데 저는 좀 다르게 봤어요. Cowork의 진짜 가치는 단일 작업이 아니라 워크플로우예요. ... 도구 하나를 잘 쓰는 게 아니라 도구들을 연결해서 시스템을 설계하는 거예요." `[09:44]`

> "비유하자면요. 자동차 부품 하나 잘 만드는 거랑 그 부품들을 연결해 자동차 한 대 굴러가게 하는 거랑 차원이 다른 일이에요." `[10:16]`

## 5단계 워크플로 (시간순)

화자가 명령 한 줄로 자동화한 파이프라인. 핵심 골격은 "수집 → 분석 → 산출물 → 저장 → 전달"이다.

> "수집, 분석, 산출물, 저[장], 전달. 이 다섯 단계 구조만 머릿속에 [박혀] 있으면 여러분만의 워크플로우를 무한히 설계할 수 있어요." `[10:39]`

| 단계 | 동작 | 사용 도구 |
|---|---|---|
| 1. 수집 | 한국경제(한경닷컴) 사이트에서 헤드라인 5건 이상 수집 | WebFetch (Cowork 기본 내장) |
| 2. 분석/작성 | 수집한 헤드라인 기반으로 브리핑 보고서 작성 | Cowork (Claude) |
| 3. 저장 | 보고서를 마크다운/Word(docx) 파일로 로컬 저장 | 파일 시스템 |
| 4. 업로드 | 생성한 파일을 구글 드라이브에 업로드 → 공유 링크 획득 | 구글 드라이브 MCP (커넥터) |
| 5. 전달 | 공유 링크를 카카오톡 "나와의 채팅"에 전송 | macOS/Windows MCP (GUI 자동 조작) |

전체 파이프라인을 **스케줄**에 등록해 매일 자동 실행.

> "이 풀 과정을 ... 스케줄로 등록을 해서 매일 오전 9시마다 ... 실행을 할 수 있도록 ... 설정을 할게요." `[06:47]`

(주의: 마무리 요약에서는 "매일 아침 8시에 자동 실행" `[11:05]`이라고 말해 오전 시각이 9시/8시로 엇갈린다. 핵심은 "매일 아침 정해진 시각 자동 실행"이며 구체 시각은 데모 설정값으로 보임.)

## 환경 셋업

### Cowork 사용 전제 조건
> "Cowork를 쓰려면 두 가지가 필요해요. 첫째는 Claude 프로 이상 구독이에요. 무료 플랜에서는 안 돼요. 둘째는 Claude 데스크탑 앱이에요. 이거 중요해요. 웹브라우저에서는 아직 안 돼요. 꼭 데스크탑 앱을 깔아야 한다는 거 기억해 주세요." `[01:37]`

- **설치**: Claude 웹사이트 → 왼쪽 아래 "앱 및 확장 프로그램 받기" 버튼 → OS(맥/윈도우)에 맞는 설치 파일 다운로드 `[01:48]`
- **메뉴**: 앱을 열면 채팅 / Cowork / 코드 메뉴가 보임. 업데이트로 위치가 바뀔 수 있으니 메뉴를 둘러보면 Cowork가 어딘가에 있음 `[01:58]`
- **작업 폴더 지정**: 바탕 화면에 새 폴더를 만들고(예: 이름을 `에이전트`로) Cowork가 일할 폴더로 지정 `[02:09]`

### MCP / 커넥터 연결
- **구글 드라이브 MCP**: Claude 앱 설정 → 커넥터 메뉴 → "구글 드라이브" 검색 → 추가 → 구글 로그인 → 권한 동의. 연결하면 Claude가 만든 파일을 바로 드라이브에 올려 공유 링크까지 받음 `[04:23]`
- **macOS MCP**: Claude가 마우스 클릭·키보드 타이핑·스크린샷을 할 수 있게 해 주는 도구. 확장 프로그램에서 "macOS MCP" 검색해 설정. 단 시스템 설정에서 **손쉬운 사용 / 화면 기록 / 입력 모니터링** 권한을 하나씩 켜야 함. 처음 한 번만 하면 됨 `[05:24]`
- **Windows 사용자**: macOS MCP 대신 **Windows MCP**를 이용해 컨트롤 `[05:42]`

### WebFetch 기능
> "WebFetch라는 기능이에요. ... 말 그대로요. Claude가 특정 웹사이트를 직접 열어서 내용을 읽어보는 기능이에요. ... URL 하나만 알려주면 Claude가 [그] 페이지 전체를 한 번에 읽어 와요." `[02:42]`

> "설치는 따로 안 해도 돼요. Claude Cowork에 기본으로 들어 있는 기능이거든요." `[03:11]`

## 팁
- **작게 시작해서 한 단계씩 붙이기**: `[의견]` 화자가 강조한 도입 방식.
  > "한 번에 다 만들려고 하면 [엎어]져요. 작은 거 하나 만들고 잘 돌아가면 거기 한 단계 더 붙이시는 식으로 가시면 됩니다." `[11:25]`
- **이 구조는 어디에든 적용 가능**: 경제 브리핑은 예시일 뿐. 경쟁사 뉴스 모니터링 → 보고서 → [불명확: 원문 "요선에 정리해 두기"], 서비스 리뷰 수집 → 감정 분석 → 슬랙 전송 등 `[10:24]`
- **카카오톡 외 대안**: 텔레그램 등으로도 같은 세팅 가능 (텔레그램은 API가 열려 있어 오히려 더 쉬움) `[09:00]`

## 보조 기능: Claude Dispatch
스케줄 자동화와 별개로, 외부에서 즉석 명령을 넣고 싶을 때 쓰는 기능.

> "Claude Dispatch라고 있는데요. 쉽게 말하면요. 핸드폰으로 내 컴퓨터의 Claude 데스크탑 앱을 다루는 기능이에요. 핸드폰이 리모컨이 되는 겁니다." `[09:18]`

- 설정: 핸드폰에 Claude 앱 설치 → 왼쪽 Dispatch 메뉴 → 명령 전송 `[09:32]`
- **화자의 주의**: `[의견]`
  > "근데 이 Dispatch가 아직 좀 많이 불안정해 가지고 그렇게까지 엄청 추천드리고 있지 않습니다." `[09:37]`
