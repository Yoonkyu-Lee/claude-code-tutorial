# conference/ — 회의(비판적 토론)·재구성 문서

`notes/`(영상 1편=노트 1편)나 `digests/`(영상 전문 paraphrase)와 달리, 이 폴더는
**여러 영상·자료를 가로질러 한 주제를 비판적으로 토론하고 내 지식으로 재구성한 원본 문서**를 모은다.

- 산출물 성격: 특정 영상의 요약이 아니라, 주제에 대한 **독립적 분석·판단·실행계획**.
- 자료(영상·zip 등)는 *계기이자 참고*일 뿐, 본문은 배경지식으로 재구성한다.
- 사실은 출처를 달고, 주관 판단은 `[의견]`으로 분리한다(노트 규칙 준용).
