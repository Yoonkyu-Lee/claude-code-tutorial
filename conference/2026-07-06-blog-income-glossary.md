## 1. 유튜버에 빗대면 — 블로그 수익 구조

블로그는 **"영상 대신 글, 추천 대신 검색"인 유튜브**다. 대응이 거의 1:1이다.

| 유튜브 (아는 것) | 블로그 (같은 것, 다른 옷) |
|---|---|
| 채널 | 블로그 / 사이트 |
| 영상 한 편 | 글(포스트) 한 편 |
| 조회수 | 방문자·트래픽 |
| **유튜브 추천 알고리즘**이 영상 물어다 줌 | **검색(구글·네이버)**이 글 물어다 줌 |
| 썸네일·제목·조회수 유지율 최적화 | **SEO**(검색 최적화) |
| 영상에 붙는 광고 수익 | 글에 붙는 **애드센스** 광고 수익 |
| "내 채널 CPM/RPM 얼마" | 똑같이 **RPM**(1,000회당 수익) |
| 더보기란 쿠팡·제휴 링크 | 글 속 **제휴 링크**(쿠팡파트너스 등) |
| 협찬·PPL 영상 | 협찬 글 |
| 멤버십·슈퍼챗·굿즈·강의 | 전자책·강의·구독 등 **자체 상품** |
| 게임채널/요리채널 (분야) | **니치**(분야) |
| 노란딱지·수익화 정지 | 저품질·**펍벤**(광고 정지) |

**결정적 차이 하나 — "발견"을 누가 해주나:**
- 유튜브는 **추천 알고리즘**이 사람을 데려온다 → 썸네일·제목·초반 이탈률이 승부.
- 블로그는 **검색**이 사람을 데려온다 → 누가 "○○ 신청 방법" 치면 내 글이 위에 떠야 한다.
  그 작업이 **SEO** = *유튜브의 썸네일·알고리즘 최적화에 해당하는 것.*

**돈 버는 4가지 방법은 유튜브와 완전히 동일하다:**
1. **광고** — 조회수(방문자)당 돈. (유튜브 광고수익도 구글 관리 = 애드센스와 같은 지갑)
2. **제휴** — 내 링크로 남의 물건 팔리면 수수료. (더보기란 쿠팡링크)
3. **협찬** — 브랜드가 돈 주고 글/영상 의뢰.
4. **자체 상품** — 내 강의·전자책·구독. (멤버십·굿즈)

> 한 줄: **블로그 부업 = 글 + 검색으로 하는 유튜브.** 뼈대는 이게 전부다.

## 2. 용어 사전

### A. 플랫폼·자산 (어디에 글을 쌓나)
| 용어 | 뜻 | 유튜브 비유 |
|---|---|---|
| **도메인** | 사이트 주소(예: `mysite.com`) | 채널 URL |
| **자체 도메인** | 내가 산 내 주소에 내 사이트를 차리는 것 | 유튜브 말고 *내가 만든 동영상 사이트*를 여는 것 — 아무도 못 쫓아내지만 사람은 알아서 모아야 함 |
| **호스팅** | 사이트 파일을 얹어두는 서버(월세 낸 땅) | — |
| **워드프레스** | 세계 표준 블로그 제작 툴(자체 도메인용) | 전문 편집 프로그램 |
| **블로그스팟/티스토리/네이버블로그** | 무료로 빌려 쓰는 남의 플랫폼 | *유튜브에 올리기* — 공짜지만 남의 땅, 쫓겨날 수 있음 |
| **포스트** | 글 한 편 | 영상 한 편 |
| **정적 사이트(SSG)** | 미리 만들어둔 HTML을 뿌리는 가벼운 사이트(Astro/Next 등) | — |

### B. 트래픽·유입 (사람을 어떻게 데려오나)
| 용어 | 뜻 | 유튜브 비유 |
|---|---|---|
| **트래픽** | 사이트에 들어온 사람 흐름 | 조회수 |
| **페이지뷰(PV)/세션/방문자** | 페이지 열람 횟수 / 방문 단위 / 사람 수 | 조회수/시청 세션 |
| **오가닉(자연 검색) 유입** | 검색으로 공짜로 들어온 트래픽 | 추천으로 뜬 조회수 |
| **SEO** | 검색 상위에 뜨게 하는 작업 | 썸네일·제목·알고리즘 최적화 |
| **키워드** | 사람들이 검색창에 치는 말 | 검색 잘 되는 영상 주제 |
| **롱테일 키워드** | 경쟁 낮은 구체적 검색어("아이폰" 대신 "아이폰15 배터리 교체 비용") | 틈새 주제 |
| **검색 의도** | 그 검색어를 친 사람이 진짜 원하는 것 | 시청 의도 |
| **크롤링/색인(인덱싱)** | 구글이 내 글을 훑어(크롤) 검색DB에 등록(색인)하는 것 | 유튜브가 영상을 처리해 검색 가능하게 만듦 |
| **백링크** | 다른 사이트가 내 글을 링크(추천)한 것 | 다른 채널이 내 영상 추천·인용 |
| **SERP** | 검색 결과 페이지(Search Engine Results Page) | 검색 결과 목록 |
| **상위노출** | 검색 1페이지 위쪽에 뜨는 것 | 떡상·알고리즘 간택 |

### C. 수익화 (트래픽을 돈으로 바꾸는 용어) ★핵심
| 용어 | 뜻 | 비고 |
|---|---|---|
| **애드센스(AdSense)** | 구글의 광고 붙이기 프로그램(달러) | 유튜브 광고수익의 블로그판 |
| **애드포스트** | 네이버판 애드센스(원화) | 단가 낮음 |
| **CPC** | Cost Per Click. 광고 클릭 1번당 광고주가 내는 돈 | 그중 일부를 내가 받음 |
| **CPM** | Cost Per Mille. 광고 노출 1,000번당 단가(광고주 관점) | Mille=1,000(라틴어) |
| **RPM** | **Revenue Per Mille.** 내 수익 기준, 1,000회당 내가 버는 돈 | 유튜버가 말하는 그 RPM과 동일. 가장 중요한 지표 |
| **광고 네트워크 사다리** | AdSense → Ezoic → Mediavine → Raptive | 위로 갈수록 RPM↑, 트래픽 문턱↑ |
| **제휴(어필리에이트) 마케팅** | 내 링크로 물건 팔리면 수수료 | 쿠팡파트너스, 아마존 |
| **전환(컨버전)** | 방문자가 실제로 구매·가입한 것 | |
| **전환율** | 방문자 중 전환된 비율 | |
| **수수료(커미션)** | 전환 1건당 내가 받는 돈 | |
| **EPC** | Earnings Per Click. 제휴 링크 클릭당 평균 수익 | 제휴 효율 지표 |
| **아비트리지** | 싸게 얻은 트래픽을 비싸게 되파는 차익거래 | 이 판의 본질 |
| **RSOC** | Related Search on Content. 콘텐츠에 연관검색을 얹는 애드센스 계열 아비트리지 포맷 | |

> **CPC / CPM / RPM 정리:** 셋 다 "Per Mille=1,000당 / Per Click=클릭당" 구조.
> CPC·CPM은 광고주가 내는 **비용(Cost)**, RPM만 내가 버는 **수익(Revenue)**. 내 통장 기준 지표는 RPM.

### D. 품질·알고리즘 (구글이 채점하는 기준)
| 용어 | 뜻 |
|---|---|
| **구글 코어 업데이트** | 구글이 순위 기준을 크게 바꾸는 정기 업데이트(순위 출렁임) |
| **Helpful Content** | "진짜 도움 되는 글만 올려라" 정책(현재 핵심 알고리즘에 상시 내장) |
| **E-E-A-T** | Experience·Expertise·Authoritativeness·Trust. 구글의 품질 채점 축 |
| **YMYL** | Your Money or Your Life. 돈·건강·법률처럼 삶에 큰 영향 주는 주제(품질 기준 더 엄격) |
| **저품질/스팸 정책** | 얇거나 AI로 양산한 글을 걸러내는 규칙(Scaled Content Abuse 등) |
| **펍벤(publisher ban)** | 광고 계정 정지 (유튜브 노란딱지·수익화 정지에 해당) |

### E. 규모·전략 (오래 살아남는 법)
| 용어 | 뜻 |
|---|---|
| **니치** | 특정 분야·주제 |
| **에버그린 콘텐츠** | 시간 지나도 계속 검색되는 글(vs 트렌드) |
| **트렌드잭킹** | 지금 뜨는 이슈를 노려 트래픽 확 뽑기(수명 짧음) |
| **프로그래매틱 SEO** | 데이터 + 템플릿으로 수천 페이지를 자동 생성하는 방식 |
| **데이터 해자(moat)** | 남이 복제 못 하는 데이터셋 = 방어선 |
| **오디언스** | 내가 소유한 팬(이메일 구독자 등). 유튜브 구독자에 해당 |

## 3. 두 축으로 다시 보기

- **"블로그스팟 대신 자체 도메인"** = "유튜브에만 올리지 말고 내 사이트도 차려 쫓겨날 위험을 줄여라."
- **"AI 양산 대신 데이터 해자"** = "남들 다 만드는 영상 말고, 남이 못 만드는 걸 만들어라."
