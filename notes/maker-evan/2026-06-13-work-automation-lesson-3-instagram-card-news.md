## 한 줄 요약

Claude Code에 `Instagram Card News Generator` 스킬을 트리거해서 웹에서 주제 관련 이미지를 긁어 오고 PPT로 합성한 뒤 Canva MCP로 업로드해 편집 가능한 카드 뉴스를 만드는 워크플로.

- 화자: 메이커 에반 본인
- 자막: 한국어 자동 자막 (타임스탬프 포함)

## 핵심 아이디어

화자가이 영상에서 보여 주는 자동화의 전체 골격:

1. GitHub CLI로 프로젝트를 원격 저장소에 연결해 두는 것이 **맨 처음 필수 작업**.
2. Claude Code에 `Instagram Card News Generator` 스킬을 설치하고 한 줄 프롬프트로 트리거.
3. 스킬이 내부적으로 (a) 웹 이미지·내용 조사 → (b) PPT 스킬로 합성 → (c) Canva MCP로 업로드.
4. Replicate API는 **선택**. 없어도 동작하지만 고퀄리티 이미지를 원하면 크레딧 충전 추천.
5. 최종 산출물은 Canva에서 직접 편집 가능한 카드 뉴스 8장.

> "이런 식으로 해 가지고 어 자동화 카드 자동화가 가능하다. 캠바 연동도 가능하다라고 보시면 될 것 같습니다." [07:56]

## 환경 셋업 (순서대로)

### 1) GitHub 회원 가입 + GitHub CLI 설치

> "일단은 먼저 하셔야 될게 뭐냐면 회원 가입을 먼저 해 주셔야 돼요. 첫 번째로 회원 가입을 해 주셔야 될 거요. 바로 기허브[GitHub]라는 곳입니다." [00:20]

- GitHub에 가입·로그인.
- GitHub CLI 사이트에 들어가 OS별(맥/윈도우) 버전 다운로드 후 설치.

### 2) 터미널에서 GitHub CLI 로그인 (Claude Code에 시키기)

> "터미널이나 파워쉐[PowerShell] 같은 데를 열어 주신 다음에 로그인을 해 달라고 하시면 됩니다. 그럴 때 어떻게 하시면 되냐면 클루드 코드[Claude Code]에게 물어보시면 돼요." [01:02]

화자가 추천하는 프롬프트:

> "기터브[GitHub] CLI 이용해서 로그인 해 줘. 웹 브라우저 통해서 로그인 할 거야" [01:13]

이렇게 요청하면 Claude Code가 인터랙티브 질문을 던지고, 웹 브라우저 인증으로 로그인이 이어진다.

### 3) Replicate API 키 발급 (선택)

> "그다음으로 세팅을 해 주셔야 될게 뭐냐면 레플리케이트[Replicate]라는 곳입니다. 이 레플리케이트라는 곳에 API 키가 필요하거든요." [01:31]

- Replicate에서 버튼을 눌러 Account Settings → API 토큰 발급.
- Billing 메뉴에서 크레딧 충전. 화자는 약 5달러 정도면 충분히 많이 쓸 수 있다고 언급:

> "한 5불 정도만 충전을 해 주시면 충분히 많이 쓸 수 있으니까 충전을 해 주시는 거를 추천드립니다. 물론 이게 없어도 쓸 수 쓰는 데는 문제는 없습니다. 근데 고퀄리티의 이미지를 만들기 위해서는 좀 더 쓰시는 거를 추천을 드리고 있습니다." [01:51]

화자는 이 영상에서 **Replicate 없는 버전**으로 시연한다.

### 4) Canva MCP 커넥터 설정

> "여기 설정에 들어가셔 가지고 커넥터에 맞춤 설정으로 들어가신 다음에 여기 커넥터의 MCP에 캔바[Canva] MCP를 설정을 하셔야 됩니다. 이거를 연결을 하시고 로그인까지 해 주셔야 돼요." [04:37]

- Claude(Code)의 설정 → 커넥터 → 맞춤 설정 → MCP에 Canva MCP 추가.
- Canva 로그인 필수.

### 5) 작업 폴더(workspace) 만들기

- "다른 폴더를 추가" → workspace에서 새 폴더(`인스타 카드 뉴스` 같은 이름) 생성 → 열기.
- 워크스페이스 신뢰 부여.

## 워크플로 (실제 시연)

### Step A: 프로젝트를 GitHub 레포에 연결

GitHub CLI 로그인이 끝난 상태에서 Claude Code에:

> "이 프로젝트 기터브[GitHub] 연동해 줘. 깃 레포지토리 초기화." [02:58]

화자의 진행 메모:

> "그러면 이 친구가 막 뭐라고 물어보면서 처리를 해 줘요. 그러면 어 저는 일단 권한 건너뛰기를 할게요. 그러면 얘가 막 기 스테이터스를 확인을 하면서 이제 기터브[GitHub]라는 곳에 연동을 합니다. 그 퍼블릭으로 하시면 되고 인스타 카드 뉴 해 가지고 이렇게 만들면 되십니다." [03:11]

**강조 반복** (화자가 두 번 말함):

> "이 작업이 맨 처음에 필요합니다. 이 작업이 맨 처음에 필요합니다." [03:34]

### Step B: 카드 뉴스 생성 스킬 트리거

설치된 스킬 목록 중 `인스타 카드 뉴스 제너레이터` 선택. 화자가 사용한 트리거 프롬프트:

> "카드 뉴스 만들어 줘. 나는 메이커 에반이고, 바이브 코딩 관련된 어 주제로 카드 뉴스 만들고 싶어. 관련 뉴스로 한번 만들어 줘 봐. 클로드 코드[Claude Code] 스킬이면 더 좋아. 카드 여덟 장 만들어 줘." [03:57]

### Step C: 스킬 내부 동작 (화자 설명 그대로)

> "이 친구의 동작 방식을 간단하게 설명을 드리면 일단 웹에서 적절한 이미지 아 적절한 내용들을 다 이제 조사를 합니다. 그리고 적절한 내용에 걸맞는 이미지들을 검색을 해 가지고 긁어 와요. 그다음에 그거를 이제 PPT 스킬을 이용을 해 가지고 합성을 해 주게 됩니다. 그다음에 그거를 캠바[Canva]에다가 업로드하는 과정을 거쳐요." [05:19]

요약 흐름:
1. 웹 이미지·내용 소싱 (`web 이미지 소싱`이라는 description으로 동작) [05:06]
2. 관련 이미지 검색·다운로드
3. PPT 스킬로 합성
4. 결과를 Canva에 업로드

화자는 스킬이 자체적으로 밸리데이션도 수행한다고 언급:

> "그 이 친구가 막 밸리데이션도 알아서 하거든요. 그 이런 식으로 이미지가 생성이 됩니다." [05:48]

### Step D: 결과 확인 + Canva 업로드

- `output` 폴더의 슬라이드를 Finder로 열어 확인.
- 화자 평:

> "저는 꽤 만족스럽네요." [06:16]

- 이어서 Canva 업로드 요청 → 업로드 완료 후 Canva에서 표/이미지 편집 가능.

> "이런 식으로 캠바[Canva]에서 표 편집 가능한 이미지들이 나오는 거를 확인을 할 수가 있습니다." [07:23]

## 팁

- **Replicate는 선택**. 없어도 워크플로는 굴러간다. 단, 이미지 퀄리티 욕심나면 5달러 정도 충전. [01:56]
- **첫 작업 = GitHub 연동**. 화자가 같은 문장을 두 번 반복할 정도로 강조. [03:34]
- **트리거 프롬프트는 자기소개 + 주제 + 분량을 명시**해서 한 문장으로 던지면 된다. (Step B 예시 참조)
- 최종 비용 모델: API 비용 없이 **구독 비용만**으로 운영 가능.

> "이 모든 거를 하는데 레플리케이트[Replicate] API를 쓰지를 않아 가지고 어 API 비용은 안 들고 이제 구독 비용만 나간다고 보시면 될 거 같아요." [07:39]

## 불명확

- `Instagram Card News Generator` 스킬의 **배포 채널·설치 방법**은 영상에서 보이지 않음. 화자가 "제가 드릴 스킬"이라고만 언급 [03:43]. 멤버십·별도 링크 어디인지 [불명확].
- "그 퍼블릭으로 하시면 되고 인스타 카드 뉴 해 가지고 이렇게 만들면 되십니다" [03:23] — 레포 이름을 `인스타 카드 뉴`(잘림)로 설정한 것인지, 전체 명을 입력한 것인지 자막상 [불명확: 원문 "인스타 카드 뉴 해 가지고 이렇게 만들면 되십니다"].
- 카드 PPT가 Canva로 올라갈 때 사용하는 **포맷·해상도·템플릿**은 영상 시연 화면이 자막에서 추출되지 않아 [불명확].
- Canva MCP의 정확한 명칭·제공자 (공식 Canva MCP인지 서드 파티인지) [불명확].
- "워크스페이스 실뢰" [02:44] = "워크스페이스 신뢰(trust)"로 보이나 자막 원문은 [불명확: 원문 "워크스페이스 실뢰"].
