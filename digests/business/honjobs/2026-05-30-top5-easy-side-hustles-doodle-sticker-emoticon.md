## 요약 (주제별)

### 이 영상이 파는 것 (전체 구조)
- 하루 15~30분(또는 1시간) 자본금 0~300원으로 "누구나" 가능하다는 **부업 5종 모음집**. 다섯 개 모두 공통 뼈대는 동일하다: **① 무료/저비용 이미지 소스를 확보 → ② 무료 웹 디자인 툴(ChatGPT·Gemini·Canva·미리캔버스·Illustrator 무료체험)로 상품 이미지 제작 → ③ POD/오픈마켓/블로그에서 판매**.
- 반복되는 세일즈 훅: "지금까지 소개된 적 없는 방법", "초블루오션", "조회수 적을 때 본 사람이 유리하니 빨리 선점하라", "자면서도 수익". (모두 화자 주장/미검증, 마케팅 톤)

### 부업 1 — 낙서 그림으로 아기 옷 POD 판매 [00:43~03:01]
- 동물을 낙서하듯 대충 그린 뒤 **ChatGPT**(자막 "채집비티")에 올려 "이 낙서체로 곰 그려줘"식 프롬프트로 완성도 높은 낙서풍 그림 생성, 배경을 PNG 투명 처리.
- **Printify**(자막 "프린티파이") 가입 → Create Product → Kids & Babies → Start Designing → 이미지 업로드 → Save Product로 아기 옷 상품 제작. 직접 제작·디자인 능력 불필요.
- 무한 복제: **Gemini**(자막 "재미나이")에 같은 낙서를 올려 "이 낙서체로 악어·낙타·기린 등 수백 동물 그려줘"로 대량 변형 → **Etsy**(자막 "엣지")에서 판매. (Etsy 판매법 자체는 별도 링크로 미룸)
- 주장: 낙서 한 장 10분 투입 → 한 달 뒤 100만원 이상, "돈을 복사하듯" 반복. (주장/미검증)

### 부업 2 — 사인펜 인물 스케치 벡터화 POD 판매 [03:01~07:37]
- 준비물은 **사인펜 하나(약 300원, 집에 있으면 0원)**. 얇은 종이를 핸드폰 화면(밝기 최대)의 인물 사진 위에 테이프로 고정하고 투과된 상을 사인펜으로 따라 그린다. "졸라맨 수준" 실력이면 충분, 선이 삐뚤어도 됨.
- **Adobe Illustrator 무료체험판**(네이버에 "어도비" 검색 → 월별 결제 선택, 7일 무료지만 다운로드 위해 결제정보 입력 필요)을 설치 → 스케치를 불러와 **이미지 추적 → 확장**으로 벡터 변환 → 그룹 풀기 → 배경 삭제 → 매끄럽게 다듬기 도구로 삐뚤어진 선 보정 → 파일 내보내기 저장.
- 결과물(인물 캐릭터 스케치)은 굿즈로 제작해 팔거나 이미지 자체로 판매. 이미지만 팔 땐 인스타그램, 실물은 POD 판매 사이트.
- 주장: 퇴근 후 30분으로 월 50만원 이상. (주장/미검증)

### 부업 3 — 배달 음식 스티커 제작·판매 [07:37~11:19]
- 배달 음식에 붙는 상호 스티커. 화자는 "몇 업체가 독점하다시피" 하는 시장이라 주장. 채널 사이트 **혼스닷컴(www.혼스.com)** 가입 → "배달 스티커 부업" 카드 → **무료 36종 스티커 파일 다운로드**.
- **Canva**(자막 "캠바")에서 맞춤형 65×65mm 캔버스 생성 → 파츠 업로드 → 무료 폰트/파츠/그리기(색칠)로 스티커 디자인 → PDF 다운로드.
- **스마트스토어**에서 판매, 주문이 오면 상호명만 바꿔 대응(스마트스토어 운영법은 별도 영상). 판매 후 인쇄는 **혼잡스 성수동 인쇄소**(혼자스 사이트 → 혼잡스 인쇄소 → 배달 스티커 제작 상품 → 주문번호+PDF 업로드)에서 주문~배송까지 대행.
- 주장: 하루 1시간, 마우스 클릭 몇 번 15분 업로드 → 월 100만~200만원. 화자 본인이 "지금도 매달 200만원 이상 이 방법으로 벌고 있다". "핵심 파이프라인", "초블루오션". (주장/미검증)

### 부업 4 — 간판/상호 캘리그라피 이미지 제작·판매 [11:19~15:45]
- 식당 간판에 들어갈 붓글씨 상호 이미지를 제작해 판매. 1차 방법: **미리캔버스**(1000×1000px)에서 텍스트에 "포천 막걸리체"(붓글씨 느낌 폰트) 적용해 "포천 돼지갈비" + 도장(인주) 요소 + 돼지 붓그림 조합.
- 한계 지적 [의견]: 포천 막걸리체는 이미 널리 알려진 폰트라 사장님들이 원치 않고 붓글씨 느낌이 약함. → 2차 방법: **혼잡스닷컴**에서 "글씨 파츠" 다운로드 후 미리캔버스에 업로드해 조합하면 "작가가 붓으로 쓴 느낌".
- 판매법: 네이버 블로그를 포트폴리오로 꾸며(블로그 디자인은 이전 영상 참고) **매일 1개씩** 캘리그라피 상호를 포스팅, **톡톡 파트너** 가입·등록해 주문 접수. 한 달 꾸준히 하면 주문 발생 주장.
- 주장: 건당 10만~30만원(처음엔 저렴하게 홍보), 1시간 등록으로 매달 10만~30만원, "인형 눈붙이기보다 쉽다". (주장/미검증)

### 부업 5 — 파츠 조합 이모티콘/스티커 제작·판매 [15:45~20:42]
- 눈·코·입 등 **파츠를 게임 캐릭터 만들듯 조합**해 이모티콘/스티커 제작. 채널 사이트 **손잡스닷컴(www.잡s.com)** 가입 → 이모티콘 부업 패키지 다운로드.
- **Canva** 가입(캔바 프로 결제 페이지는 "나중에 하기") → 파츠 업로드 → 맞춤형 740×640px(네이버 이모티콘 규격) → 파츠 조합·색칠·레이어 이동으로 스티커 제작 → PNG 저장. 개성 강한 이모티콘은 어려워, **블로그 포스팅용 스티커**(리뷰 단락 구분·"물품 제공받음" 멘트용)가 수요 많아 판매 유리하다고 안내(네이버 OGQ 마켓 참고).
- 등록: 아이콘 24개 제작(하나는 240×240px, 96×74px로도 별도 제작) → **네이버 마켓(OGQ)** 크리에이터 등록 → "멈춰 있는 스티커"로 제목·태그·번호별 업로드, 메인 240px·탭 96×74px 지정.
- 정산: 약 2주 승인, 매출 1개월 정산, **수수료 제외 70%가 수익**. (수익률만 사실적 언급, 판매량은 미검증)

### 세일즈 구조 (유입 장치 — 정보로 기록)
- 5개 부업 전부 채널 자체 사이트로 유입시킨다: **혼스닷컴**(배달 스티커 무료 36종), **혼잡스닷컴**(글씨 파츠), **손잡스닷컴/www.잡s.com**(이모티콘 부업 패키지), 그리고 **혼잡스 성수동 인쇄소**(발주·배송 대행). 파일·소스는 무료 제공을 표방하나 가입(이메일)이 요구되며, 인쇄 발주는 채널 인쇄소로 연결된다.
- 반복 CTA: "빨리 선점하라", 구독·좋아요, "어려운 부업을 쉽게 기획해 계속 전달하겠다".

### 주의·한계 (영상이 명시하지 않은 리스크)
- 영상은 **판매 성사·수요·플랫폼 승인**을 실증하지 않는다. 모든 금액은 화자 주장.
- 저작권/초상권 이슈: 부업 2는 "좋아하는 연예인 사진"을 따라 그리라고 안내 → 초상권·저작권 리스크가 있으나 영상은 언급 안 함 (`[불명확]`, 화자가 다루지 않음).
- Illustrator 무료체험은 결제정보 입력이 필요하고 7일 뒤 과금될 수 있음(영상도 "결제 후 취소해도 7일 이용 가능"이라 언급).

## 전문 — 시간순 워크스루

### [00:00–00:43] 오프닝 훅 — "낙서 한 장으로 월 100만원"
낙서 그림 한 장을 10분 만에 올려두면 한 달 뒤 100만원 이상 벌 수 있고, 지금도 많은 사람이 "자면서도" 수익을 낸다고 주장하며 시작한다(주장/미검증). 아직 거의 알려지지 않은 블루오션이며 하루 10분씩 하나씩 등록하면 자본금 0원으로 큰 수익도 가능하다고 강조.
> "이 영상 조회수가 적을 때 보신 분들은 꼭 따라해 보시고", "이 영상이 올라가고 나서는 앞으로 여기저기 채널에서 이 방법이 소개될 수 있으니까 빠르게 선점하시기 바랍니다." [00:29][00:38]
(전형적 선점 압박 CTA — 마케팅 톤, 정보 가치 없음)

### [00:43–00:59] 부업 1 소개 — 낙서로 아기 옷 만들어 팔기
첫 번째 방법은 낙서 그림 한 장으로 **아기 옷을 제작·판매**하는 것. 내가 직접 제작하지 않아도 되고 디자인 능력이 없어도 가능하며, 낙서 한 장을 "돈을 복사하듯" 반복해 확장하는 방법까지 알려주겠다고 예고.

### [00:59–01:11] 낙서 그리기 + 파일 준비
동물을 진짜 낙서하듯 대충 하나 그린다. 어려우면 화자가 남긴 링크에서 낙서 그림을 다운로드해 써도 된다. 촬영한 사진을 컴퓨터로 옮기고 **ChatGPT**(자막 "채집비티"의 오인식)에 접속.

### [01:11–01:33] ChatGPT로 낙서풍 그림 생성 + 배경 제거
ChatGPT에 낙서 사진을 올리고 "이런 낙서 그림체로 네가 창작을 해서 낙서한 듯한 곰을 그려줘"라고 프롬프트를 넣으면 낙서 같으면서도 완성도 높은 그림이 나온다. 이어 "이 그림의 배경을 PNG 파일로 해주고 누끼를 따줘"라고 하면 배경 없는 투명 파일이 만들어진다.

### [01:33–02:19] Printify로 아기 옷 상품 만들기
구글에 **Printify**(자막 "프린티파이") 검색·접속 → Get Started → 구글 아이디 로그인 → 나오는 6개 질문은 중요치 않으니 모두 첫 번째 답 클릭 → Create Product 버튼 → **Kids & Babies** 선택 → 해당 제품 클릭 → Start Designing → 왼쪽 바 Upload로 아까 만든 이미지 삽입(인쇄 가능 영역에 자동 업로드) → Save Product로 상품 하나 완성.

### [02:19–02:49] 무한 복제 — Gemini로 동물 대량 생성
이 상품을 "무한대로 복사"하기 위한 응용법: 구글에 **Gemini**(자막 "재미나이") 검색·접속 → 내가 만든 낙서 그림을 올리고 "이 낙서 그림체 그대로 악어 그려줘, 낙타 그려줘, 기린 그려줘" 식으로 수백 종 동물을 요청. 그린 게 동물이 아니라 식물이나 추상적 그림이어도 된다.

### [02:49–03:01] Etsy 판매 (링크로 미룸)
이렇게 만든 상품은 **Etsy**(자막 "엣지")에서 판매해 수익을 낸다. 다만 Etsy 판매 방법까지 다루면 영상이 너무 길어지니 그 과정은 링크로 남기겠다고 함. (판매 실전은 영상 밖으로 이월 — 세일즈 퍼널의 전형)

### [03:01–03:33] 부업 2 소개 — 사인펜 하나로 월 50만원(주장)
두 번째 부업. "사인펜 하나만 있으면 퇴근 후 30분으로 월 50만원 이상 벌 수 있다"고 주장(주장/미검증). 구매할 것은 사인펜 하나뿐이고, 비슷한 다른 방법들은 초기 투자비가 있지만 이 방법은 **사인펜 300원**만 있으면 되고 집에 사인펜이 있으면 비용이 전혀 없다고 강조.

### [03:33–03:51] 방법 요지 — 사인펜으로 그려 판다
사인펜으로 그림을 그려 판매하는 것. 그림 실력이 없어 "졸라맨밖에 못 그린다" 해도 가능하며, 기존 방법들은 돈이 많이 드는데 이건 돈이 전혀 들지 않는 방법이라고 함.
