# 복사붙여넣기로 월 500번다는 부업? 과연 사실일까? 할까말까 시리즈

> ⚠️ 검증 배너: 이 영상의 "월 300~500만 원", "10분/일 투자", "10개까지 무료 저작권 음악" 등 수익·조건 주장은 **모두 화자 주장이며 미검증**이다. 실제 수익·저작권 귀속·플랫폼 정책 준수 여부는 영상만으로 확인되지 않는다. 아래 본문에서 (주장/미검증)·[의견]으로 분리 표기했다.

## 요약 (주제별)

### 1. 영상의 프레임 — "할까말까"를 표방한 방법 소개
- 채널의 "할까말까 시리즈" 포맷으로, "복사붙여넣기만으로 월 300~500만 원"이라는 부업을 "할지 말지 정해주겠다"고 열지만, 실제 내용은 **비판·검증이 아니라 그 부업(힐링 영상 제작)을 직접 따라 하는 방법 소개**에 가깝다.
- 화자는 자기가 직접 운영할 부업 채널까지 이 영상에서 공개한다.

### 2. 수익·근거 주장 (전부 화자 주장/미검증)
- "복사붙여넣기만으로 월 300~500만 원을 벌 수 있다"는 부업 (주장/미검증).
- "대부분의 유튜브 채널이 소개하는 방법으로는 월 300~500은 불가능에 가깝다" — 화자는 자기 방법이 더 수익이 된다고 주장 (주장/미검증, 근거·실적 데이터 제시 없음).
- "수년간의 부업 경험"을 근거로 든다 (주장/미검증).
- "하루에 10분 정도만 투자하면 된다" (주장/미검증).
- 구체적 수익 인증·정산 화면·채널 지표는 영상에 등장하지 않는다.

### 3. 실전 절차 — 힐링 영상 3단계 제작
- **1단계 음악(AI 작곡)**: 네이버에서 AI 음악 제작 사이트(자막상 "키니" [불명확])를 검색·접속 → SNS 로그인 → 좌측 "파트너" 메뉴 클릭 → 장르 선택 → 컨셉에 맞춰 조성/빠르기(느림)/길이(4분 50초) 설정 → 자장가 떠올리며 멜로디 입력 → 생성. "10개까지 무료로 저작권을 소유한 음악 제작 가능" (주장/미검증).
- **2단계 배경(AI 이미지)**: 이미지 생성 사이트(자막상 "라코"→**Lasco/라스코** 추정) 접속 → 이미지 설명 프롬프트 입력(예시는 영상 더보기란 제공) → "라스코 애니메" + 스튜디오 지브리 스타일 선택 → 상세 설정에서 16:9 비율 선택 → 생성된 4개 중 택1 → 업스케일 → 다운로드.
- **3단계 편집(CapCut)**: 앱스토어/구글플레이에서 CapCut 설치 → 음악·이미지 삽입 → 배경 이미지를 1시간 길이로 늘리고, 음악 10개를 복사붙여넣기로 1시간 분량으로 만듦 → 상단 이펙트에서 "내추럴" 스타일 적용 → 익스포트.

### 4. 도구·자동화 요소
- AI 음악 생성 사이트(키니 [불명확]), AI 이미지 생성 사이트(라스코/Lasco 추정), 편집은 CapCut.
- 핵심 반복 작업은 "복사붙여넣기"(음악 10곡을 이어 붙여 1시간 롱폼화)와 프롬프트 복붙.
- 컨셉 설정을 강조: 힐링 채널이라도 "일하며 듣기 / 생각하며 듣기 / 자면서 듣기" 중 방향을 명확히 잡아야 한다는 것이 화자의 핵심 조언 [의견]. 화자는 "자기 전에 듣기 좋은 음악" 컨셉을 택함.

### 5. 채널 공개·운영·알고리즘 팁
- 화자가 만든 첫 힐링 영상이 올라간 자기 채널을 공개: 유튜브 검색 "@드림토리스"([불명확], 자막 "골뱅이 드림 토리스") → 라이브 영상 클릭 → 라이브 채팅에서 제작 방법 실시간 Q&A를 하겠다고 안내.
- [의견/주장] 힐링 영상은 초반에 알고리즘 선택을 받기 어려워, 상대적으로 노출이 잘되는 **라이브 송출**을 많이 활용한다 → 라이브 운영이 수익에 더 가까워지는 요령이라고 주장.

### 6. 세일즈·유입 구조
- 영상 더보기란에 이미지 프롬프트 "예시"를 제공(자료 배포).
- 자기 부업 채널(@드림토리스 추정)로 유도하고, 라이브 채팅으로 실시간 상담·유입을 유도하는 구조.
- 강한 유료 결제·단톡방 유입 장치는 자막상 명시되지 않음(무료 도구 위주 소개).

### 7. 결론·CTA
- [의견] 이 간단한 방법으로 꾸준히 올려보면 "감이 잡히고 힐링 영상 생태계를 이해"하게 되어 더 좋은 기획 아이디어가 나온다.
- 하루 10분 투자면 되니 꾸준히 하며 더 좋은 방법을 찾아 소개하겠다(주장). 구독·좋아요 요청으로 마무리.

## 3줄 오리엔테이션
- "복사붙여넣기로 월 300~500만 원"이라는 자극적 훅으로 열지만, 실제로는 **AI 음악 + AI 지브리풍 이미지 + CapCut으로 1시간짜리 힐링 영상을 찍어내는 방법**을 따라 하는 튜토리얼이다.
- 화자는 자기 부업 채널(@드림토리스 추정)을 공개하며, 라이브 송출로 초반 알고리즘 노출을 뚫는 요령까지 제시한다.
- 수익·조건 주장은 근거 없이 반복되는 화자 주장(미검증)이며, 하루 10분·꾸준함을 강조하는 전형적 부업 소개 톤이다.

## 전문 — 시간순 워크스루

### [00:00–00:18] 훅: "월 300~500 복사붙여넣기 부업, 할지 말지 정해준다"
"복사붙여넣기만으로 월 300만 원에서 500만 원을 벌 수 있다"는 부업을 할지 말지 딱 정해주겠다고 연다(주장/미검증). 그 부업은 바로 "힐링 영상 부업"이며, 화자는 이 부업이 "정말 대박"이라고 표현한다 [의견]. 자기가 직접 운영하게 될 부업 채널도 이 영상에서 공개할 예정이니 참고하라고 예고한다.

> "복사 붙여넣기만으로 월 300만 원에서 500만 원을 벌 수 있다고 하는 부업, 과연 할지 말지 딱 정해 드리겠습니다." [00:00]

### [00:18–00:30] "대부분의 유튜브 방법으론 불가능" — 차별화 주장
대부분의 유튜브 채널이 소개하는 방법으로는 월 300~500이 "불가능에 가깝다"고 단정한다(주장/미검증). 대신 자신은 "수년간의 부업 경험"으로 실제 수익이 될 수 있도록 자세한 방법을 소개하겠다고 한다(주장/미검증, 실적 근거는 제시 안 함).

### [00:30–00:48] 일반 방법 영상 vs 자기 방법 영상 비교
대부분의 유튜버가 소개하는 방법으로 만든 영상과, 화자가 소개할 방법으로 제작된 영상을 화면으로 비교해 보이며 "둘 중 어떤 것이 더 수익이 될 것 같냐"고 시청자에게 묻는다. 그런 영상을 어떻게 만드는지 지금부터 자세히 알려주겠다고 이어간다.

### [00:48–01:24] 1단계 음악(AI 작곡) — 사이트 접속과 초기 설정
첫 단계는 음악 준비다. "돈이 되는 힐링 채널들은 음악을 직접 작곡한다"며, 우리는 직접이 아니라 **AI로 작곡**한다고 한다. 네이버에 "키니"([불명확: 원문 "키니이"])라고 검색해 AI 음악 제작 사이트에 접속하고, 화자는 "가장 쉽고 간편하다"고 평한다 [의견]. 로그인 버튼으로 간단히 SNS 로그인을 하고, 좌측 바에서 "파트너"를 클릭하면 장르 선택 화면이 나온다(여기서 화자는 특정 파일/장르를 선택).

### [01:24–02:03] 컨셉 잡기와 곡 파라미터 설정
"수익이 되는 채널의 핵심은 컨셉"이라고 강조한다 [의견]. 같은 힐링이라도 "일하면서 듣기 / 생각하면서 듣기 / 자면서 듣기"인지 컨셉을 명확히 잡아야 한다며, 화자는 "자기 전에 듣기 좋은 음악"을 컨셉으로 채널 방향을 잡는다. 그에 맞춰 조성은 [불명확: 원문 "maor"], 빠르기는 "느림", 길이는 직접 4분 50초로 설정한다. 기본 멜로디는 "자장가를 생각하면서" 눌러 입력하면 된다고 안내한다.

### [02:03–02:17] 곡 생성과 "무료 저작권" 주장
생성하기 버튼을 누르면 음악이 완성되고 "이건 이제 내 음악"이라고 표현한다. "10개까지는 무료로 자신이 저작권을 소유한 음악을 만들 수 있다"고 주장한다(주장/미검증 — 실제 라이선스 조건은 확인 필요).

### [02:17–03:00] 2단계 배경(AI 이미지) — 라스코/지브리풍 생성
이제 힐링 영상 배경을 AI로 만든다. 웹사이트 "라코"([불명확: 원문 "라코/라스코", **Lasco** 추정)에 접속해 원하는 이미지 설명(프롬프트)을 입력하는데, 예시 프롬프트는 영상 더보기란에 남겨 두었다고 한다. 복사붙여넣기로 프롬프트를 넣으면 이미지가 생성되고, 화자는 지브리풍 채널을 만들 예정이라 "라스코 애니메"를 고르고 아래에서 "스튜디오 지브리 스타일"을 선택한다. 상세 설정에서 16:9 비율([불명확: 원문 "16대 99"])을 택하고 다시 제너레이트하면 기본 4개 이미지가 나온다.

### [03:00–03:10] 이미지 선택·업스케일·다운로드
생성된 4개 중 하나를 클릭해 선택하고, 업스케일을 눌러 화질을 올린 뒤 완료되면 다운로드 버튼으로 저장한다. 이로써 배경 음악과 배경 이미지 제작이 끝났다고 정리한다.

### [03:10–03:52] 3단계 편집(CapCut) — 복사붙여넣기로 1시간 롱폼화
편집 툴은 CapCut을 쓴다. 앱스토어나 구글플레이에서 설치·실행한 뒤, 만든 음악과 이미지를 넣는다. 올린 음악·영상을 아래 트랙으로 드래그해 옮기고, 배경 이미지를 1시간 길이로 끌어 늘린다. 음악은 **10개를 복사붙여넣기해 1시간 분량**으로 만든다. 상단 좌측 이펙트에서 화자는 "내추럴"([불명확: 원문 "내추럴해"]) 스타일을 골라 드래그로 영상에 적용한다.

> "10개의 음악을 복사 붙여넣게 하여서 1시간 분량으로 만들어 주고..." [03:36]

### [03:52–04:01] 익스포트
완성되면 우측 상단 익스포트를 클릭하고, 창이 뜨면 다시 익스포트를 눌러 영상 제작을 마무리한다.

### [04:01–04:22] 자기 채널 공개와 라이브 Q&A 안내
이제 자기 채널을 공개한다. 유튜브 검색에 "@드림토리스"([불명확: 원문 "골뱅이 드림 토리스"])를 검색하고 라이브 부분에서 해당 영상을 클릭하라고 안내한다. 이 영상이 화자가 만든 첫 힐링 영상이며, 라이브 채팅에서 제작 방법 관련 궁금증을 실시간으로 물어보면 바로 답해주겠다고 한다.

### [04:22–04:50] 꾸준한 운영과 라이브 송출 알고리즘 팁
채널을 꾸준히 운영해 더 많은 노하우를 공유하겠다고 하며, 시청자에게도 자기 채널을 참고해 영상을 올려보라고 권한다. [의견/주장] 힐링 영상은 초반에 알고리즘 선택을 받기 쉽지 않아, 상대적으로 노출이 잘되는 **라이브 송출**을 많이 활용한다며, 이 점을 참고해 운영하면 수익에 더 가까워질 수 있다고 한다.

### [04:52–05:16] 결론·CTA
[의견] 이 간단한 방법으로 일단 꾸준히 올려보면 감이 잡히고 "힐링 영상의 생태계"를 느끼게 되어, 기획에 더 좋은 아이디어가 떠오른다고 마무리한다. "하루에 10분 정도만 투자하면 되기 때문에" 꾸준히 하며 조만간 더 좋은 방법을 찾아 소개하겠다고 한다(주장/미검증). 영상이 마음에 들면 구독·좋아요를 부탁하며 끝낸다.
