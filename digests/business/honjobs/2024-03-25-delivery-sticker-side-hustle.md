# 하루 30분 만들고 부업으로 월 100만원 버는 확실한 부업(혼잡스 핵심 파이프라인 공개)

> 검증 배너: 이 영상은 혼잡스(honjobs)가 자사 자료·멤버십·인쇄소를 홍보하는 수익/부업 홍보성 콘텐츠다. "하루 한 시간 → 월 100만원", "초 블루오션", "확실히 돈이 되는 부업" 등 모든 수익·시장 주장은 **화자 주장이며 미검증**이다. 실제 수익·판매량·경쟁 강도에 대한 객관 증거는 영상에 제시되지 않는다.

## 요약 (주제별)

### 부업 아이템 — 배달 스티커
- 배달 음식에 붙어 오는 "배달 스티커"를 디자인해 판매하는 부업. 화자는 배달 사업이 예전만 못해도 매달 요리를 시켜 먹는 사람이 전국 수십만 명(주장/미검증)이라 수요가 있고, "단 몇 군데 업체가 스티커 판매를 독점하다시피" 해서 "초 블루오션 꿀 부업"이라고 주장(미검증)한다.

### 수익·시장 주장 (전부 미검증)
- "하루 날 잡고 한 시간 해서 매달 100만 원 이상"을 벌 수 있다고 주장. [의견] "정말 확실히 돈이 되는 부업"이라는 표현은 화자 주관.
- "현재 특정한 사람들만 알고 하는", "다른 사람들보다 빠르게 내일부터 시작"할 수 있다는 희소성·선점 프레이밍. 근거 데이터는 없음.

### 실전 절차 (5단계)
1. 혼잡스 사이트 접속 → 가입 → 메인에서 "배달 스티커 부업" 카드 클릭 → 무료 36종 스티커 파일 다운로드.
2. 캔바(Canva) 가입 → 맞춤 크기 **가로 65 × 세로 65mm**로 새 디자인 생성 → 다운로드한 파일 전체 업로드.
3. 원형 도구(대지) 파일을 기준으로 올리고, 캔바의 파츠·무료 폰트·그리기(색칠) 도구로 스티커 디자인 제작.
4. 원형 대지 파일은 지운 뒤 공유 → PDF로 다운로드.
5. 스마트 스토어에 등록·판매 → 주문이 오면 상호명만 바꿔 판매.

### 인쇄·배송 (혼잡스 인쇄소)
- 판매되면 혼잡스가 보유(제휴) 중이라는 성수동 인쇄소로 발주. 혼잡스 사이트 → "혼잡스 인쇄소" → "배달 스티커 제작하기" 상품 클릭 → 주문 옵션 선택 → 주문 완료 후 파일 업로드 버튼에 주문 번호 입력 + 캔바에서 받은 PDF 업로드.
- 주문부터 배송까지 혼잡스 제휴 인쇄소가 처리해 이후 신경 쓸 부분이 없다고 안내.

### 도구·진입장벽
- "디자인을 전혀 못 하고 컴퓨터를 전혀 못 하더라도" 따라 할 수 있다고 강조. 사용 도구는 혼잡스 제공 무료 파일 + 캔바(무료 폰트·파츠) 조합.
- 스마트 스토어 운영법은 별도 영상 참고를 안내.

### 세일즈 구조 — 멤버십 (월 12,000원)
- 멤버십 가입 시: 자료 지속 업데이트, 블로그·스마트 스토어 교육 자료, 상세 페이지 제작용 파츠 자료 "무한 제공", 스마트 스토어 상세 페이지 샘플, 배달 스티커 추가 파트, 블로그 홍보 방법, 추가 시안, 판매 지원 등을 제공한다고 반복 안내.
- 멤버십이 아니어도 무료 36종 파일로 영상을 따라 진행 가능하다고 함. [의견] 영상 전반에 멤버십 가입 유도가 반복되는 홍보 구조.

### CTA
- 마지막에 "구독과 좋아요" 요청. 영상 곳곳에서 멤버십 가입을 반복 요청.

## 3줄 오리엔테이션
- 혼잡스가 미는 부업은 "배달 음식에 붙는 스티커"를 캔바로 디자인해 스마트 스토어에서 팔고, 혼잡스 제휴 성수동 인쇄소로 발주·배송까지 넘기는 구조다.
- 혼잡스가 무료 스티커 소스(36종)와 유료 멤버십(월 12,000원, 추가 자료·판매 지원)을 파는 것이 핵심 사업 모델이며, 영상은 그 유입 콘텐츠다.
- "하루 한 시간 월 100만원·초 블루오션" 같은 수익·시장 주장은 근거 없이 제시된 화자 주장(미검증)이다.

## 전문 — 시간순 워크스루

### [00:00–00:21] 훅 — "하루 한 시간, 월 100만원 확실한 부업"
화자는 "하루 날 잡고 딱 한 시간 해서 매달 100만 원 이상 벌 수 있는 확실한 방법"이라며, 자신이 지금 핵심적으로 수익을 만들고 있는 파이프라인이라고 소개한다(주장/미검증). "특정한 사람들만 알고 하는 꿀 부업이라 초 블루오션"이라는 희소성 프레이밍과 함께, 당장 내일부터 시작할 수 있는 자료도 제공하겠다고 예고한다.
> "하루 날 잡고 딱 한시간 해서 매달 100만 원 이상 벌 수 있는 확실한 방법입니다" [00:00]

### [00:21–00:37] 아이템 정체 — 배달 스티커
방법은 배달을 시키면 오는 "스티커"라고 밝힌다. 배달 사업이 예전만 못해도 전국에서 매달 요리를 시켜 먹는 사람이 수십만 명에 달하고(주장/미검증), 단 몇 군데 업체가 이 스티커 판매를 "독점하다시피" 하기 때문에 "정말 콜(꿀) 부업"이라고 주장한다.

### [00:37–00:57] 진입장벽 + 혼잡스 사이트 접속
이 부업엔 어느 정도 디자인이 필요하지만, "디자인을 전혀 못 하고 컴퓨터를 전혀 못 하더라도" 할 수 있는 방법을 알려주겠다고 한다. 실행은 사이트 접속부터: 자막에는 `www.hj.com`으로 나오나 문맥상 혼잡스 사이트다 [불명확: 원문 "www.hj.com" — 정확한 도메인은 영상에서 불명확]. 가입 후 메인 페이지에서 "배달 스티커 부업" 카드를 클릭한다.

### [00:57–01:29] 다운로드 옵션 + 멤버십 첫 안내
카드를 누르면 두 가지 파일 다운로드 방식이 보인다. 화자는 지속적으로 좋은 자료를 주기 위해 멤버십을 준비 중이라며, 가입 시 자료 지속 업데이트, 블로그·스마트 스토어 교육 자료, 상세 페이지 제작용 파츠 자료를 "무한 제공"하는 "한 달 12,000원 정도의 멤버십"이라고 소개한다. 다만 멤버십이 아니어도 무료 제공 파일로 도전 가능하며, 무료 이용자는 "무료 36종 스티커 파일"을 선택해 다운로드하면 된다고 안내한다.

### [01:29–02:05] 캔바 세팅 — 65×65mm 캔버스
파일을 받은 뒤 네이버에 "캔바"(Canva)를 검색해 접속한다 [불명확: 자막 원문 "캔버라"·"캠바"는 Canva 오인식으로 교정]. 가입 후 시작하기 → 디자인 만들기 → 맞춤 크기 버튼을 눌러 **가로 65 × 세로 65mm**를 기재하고 새 디자인을 만든다. 캔버스가 뜨면 왼쪽 바의 업로드 항목 → 파일 업로드로, 앞서 혼잡스에서 받은 파일들을 모두 업로드한다.

### [02:05–02:37] 디자인 제작 → PDF 내보내기
원형 도구(대지) 파일을 불러 올리고, 캔바가 제공하는 다양한 파츠와 무료 폰트로 배달 스티커 디자인을 만든다. "그리기" 항목으로 색칠까지 자유롭게 가능해 여러 스타일을 만들 수 있다고 설명한다. 디자인을 마쳤으면 처음 불러온 원형 대지 파일은 지운 뒤, 공유 버튼 → 다운로드 → PDF 파일로 내려받는다.

### [02:37–02:57] 스마트 스토어 판매 — 상호명만 교체
이렇게 만든 파일을 스마트 스토어에서 판매하고, 주문이 들어오면 "상호명 입력 부분에 상호명만 바꿔서" 판매하면 된다고 한다. 스마트 스토어 운영 과정은 별도 영상을 참고하라고 안내하며, 멤버십 가입 시 스마트 스토어 상세 페이지 샘플과 배달 스티커 추가 파트를 받을 수 있다고 다시 언급한다.

### [02:57–03:16] 멤버십 재안내 + 판매 이후 단계
멤버십 가입 시 블로그를 통한 홍보 방법 등도 추가로 받을 수 있다며 재차 가입을 권한다. 다만 멤버십이 아니어도 영상만으로 충분히 따라 할 수 있으니 상황에 맞게 선택하라고 덧붙인다. 판매가 되면 이제 공장에 인쇄 발주를 하면 된다고 넘어간다.

### [03:16–03:59] 혼잡스 제휴 성수동 인쇄소로 발주
인쇄 발주는 혼잡스가 보유(제휴) 중이라는 성수동 인쇄소로 간편히 가능하다고 한다. 혼잡스 사이트 접속 → "혼잡스 인쇄소" 클릭 → "배달 스티커 제작하기" 상품 클릭 → 주문 들어온 상품의 옵션 선택 → 주문 완료 후 다시 상품에 접속하면 파일 업로드 버튼이 있어, 여기에 주문 번호를 입력하고 캔바에서 받은 PDF를 올리면 완료된다. 주문부터 배송까지 모두 혼잡스 제휴 인쇄소가 진행하므로 이후 신경 쓸 부분이 전혀 없다고 강조한다.

### [03:59–04:26] 클로징 + CTA
화자는 "쉽고 재미있게 진행할 배달 스티커 부업"을 소개했다며, "정말 확실히 돈이 되는 부업이기 때문에 꼭 도전해 보라"고 마무리한다(주장/미검증). 멤버십 가입 시 앞으로도 추가 시안 제공과 판매 지원을 하겠다며 다시 가입을 권하고, 구독과 좋아요를 요청하며 끝낸다.
> "정말 확실히 돈이 되는 부업이기 때문에 한번 꼭 도전해 보시기 바라고" [04:05]
