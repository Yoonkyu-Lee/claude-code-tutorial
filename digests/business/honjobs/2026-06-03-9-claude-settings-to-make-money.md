# 8분안에 클로드로 돈벌리는 세팅 9가지 따라하기

## 요약 (주제별)

### 이 영상의 전체 골자
- Claude를 "고급 구글 검색"처럼만 쓰는 사람이 많은데, 제대로 세팅하면 생산성을 크게 올리고 수익화도 가능하다는 주장. Claude 세팅 팁 9가지를 순서대로 소개한다.
- 화자는 자신이 "지금까지 클로드에 500만 원 넘게 썼고, 클로드로 만든 앱으로 수익화에 성공했다"고 주장(미검증). 자신이 만든 생산성 도구 코드를 무료로 준다고 안내(세일즈/유입 장치).

### 데이터 이관·초기 세팅 (팁 1)
- **다른 AI 데이터 통합**: ChatGPT·Gemini에서 쓰던 설정값·대화 기록을 Claude로 옮기는 기능. 절차 — Claude AI 접속 → 좌측 하단 내 아이디 클릭 → 설정 → 기능 → "가져오기 시작" 클릭 → 제공되는 프롬프트 복사 → 쓰던 다른 AI에 붙여넣어 내 정보 요약 받기 → 그 요약을 다시 복사해 Claude 창의 해당 칸에 붙여넣고 "메모리에 추가"하면 완료. 화자 주장 소요시간 2분.

### 모델 선택과 토큰 관리 (팁 2)
- Claude에는 여러 모델이 있고 모델마다 쓰이는 토큰 양이 다르다. 화자의 용도별 추천[의견]: 잡담·시간 때우기는 **하이쿠(Haiku)**, 검색·간단한 서류 작업은 **소넷(Sonnet)**, 회사 업무·숙제·앱/서비스 기획은 **오퍼스(Opus)**.
- 단 오퍼스는 프로 구독이어도 중간중간 사용 제한이 걸릴 수 있다. 화자가 든 비유: "하루 사용량이 1000이면 100을 쓰고 1~3시간 정도 쉬어야 다음 100을 쓰는 식"— 화자가 "정확한 숫자가 아니라 이해를 돕기 위한 비유"라고 명시.
- 모든 대화에 오퍼스를 쓰면 유료 구독을 하고도 제대로 못 쓴다. 화자는 이 때문에 "수십만 원을 날린 적이 있다"고 주장(미검증)하며 주의를 당부.

### 아티팩트·인수인계·프로젝트 (팁 3~4)
- **팁3 아티팩트(Artifacts) 생산성 도구화**: 새 대화창은 기존 대화와 느낌이 달라질 수 있으니 자주 하는 작업을 미리 도구로 만들어 두라. Claude 사이트 좌측 아티팩트 버튼으로 도구를 빠르게 제작. 화자는 영수증 관리·QR코드 만들기·웹사이트 즐겨찾기 정리 도구를 만들어 쓴다고 함. 이 도구 코드를 링크로 무료 제공한다고 안내(세일즈/유입 장치). 사용법 — 새 아티팩트 → 처음부터 시작하기 → 받은 코드 파일 드래그해 넣고 "이대로 도구 제작해줘".
- **팁4 인수인계 자료 + 프로젝트 + 시스템 프롬프트**: 한 채팅방에서 작업이 너무 많으면 사이트가 느려지고, 이미지를 100개 넘게 넣으면 이후 업로드가 안 됨(요금제와 무관하다고 주장). 대비책 — 채팅 시작 시 "작업 끝나면 인수인계 자료 만들어줘"라고 미리 말해두면 종료 후 인수인계 자료를 생성, 새 채팅방에 넣으면 같은 환경으로 대화를 이어갈 수 있음. 좌측 프로젝트 버튼으로 같은 작업 채팅방을 모아두면 편리(단 프로젝트에 넣어도 자동 인수인계는 안 됨). 추가 팁 — 시스템 프롬프트를 만들어 지침 박스에 붙여두면 통일된 방식으로 채팅방 사용 가능.

### 브라우저·데스크톱 연동 (팁 5~6)
- **팁5 크롬 확장 프로그램**: 브라우저를 Claude로 조작하는 기능. 크롬 웹스토어에서 "클로드"(Claude for Chrome 계열) 검색 후 설치. 구글 스프레드시트 작업 시 엑셀 기능을 일일이 검색할 필요 없이 도움받거나, 화면 캡처·검색어가 생각 안 날 때 대충 적으면 단어를 유추해주는 등 비서처럼 활용. 유료 플랜부터 사용 가능.
- **팁6 Claude 데스크톱(내 컴퓨터 안의 클로드)**: 화자가 "진짜 미친 기능"[의견]이라 강조. 구글에 클로드 검색 → 다운로드 → 설치. 파일/폴더 정리를 시켰더니 "무려 70GB 정도를 확보했다"고 주장(미검증). 별도 정리 프로그램은 악성코드가 딸려오거나 비용이 드는데 이를 Claude가 대신 해준다는 논리.

### 자동화·개발·디자인 (팁 7~9)
- **팁7 예약 작업(스케줄)**: 데스크톱 Claude에 "스케줄"이라고 입력하면 입력한 작업을 예약 작업으로 등록. 컴퓨터가 켜져 있으면 매일 체크리스트대로 작업 예약, 아침마다 뉴스 요약, 바탕화면에 쌓인 작업 파일을 정해진 시간에 특정 폴더로 정리 등 "비서"처럼 활용 가능.
- **팁8 Claude Code(본격 개발)**: 간단한 도구는 Claude로도 되지만 본격적 코딩으로 서비스를 만들려면 Claude Code가 필요. "개발자 한 명 고용"에 비유[의견]. 사용법 두 가지 — 컴퓨터에서 앱처럼 쓰기 / 터미널로 쓰기. 본격 수익화를 위해 꼭 익혀두라 권함. 자세한 내용은 다른 영상에서 다룬다고 예고.
- **팁9 디자인 기능 + 피그마 연동**: Claude에 새로 나온 디자인 기능은 특히 웹 디자인에 특화. 단 Claude는 디자인 전문 툴 출신이 아니므로 피그마(Figma)를 함께 쓰면 "금상첨화"[의견] — 피그마로 초기 디자인을 잡고 이후 디테일을 Claude로 손보면 고퀄리티. Claude 디자인은 현재 유료 플랜 이상에서 미리보기 단계로만 사용 가능하므로 피그마를 커넥터로 연결해 함께 쓰는 것이 베스트라고 권함.

### 세일즈·마무리 구조
- 무료 제공 장치: 화자가 만든 생산성 도구(영수증 관리·QR코드·즐겨찾기 정리)의 코드를 링크로 무료 배포한다고 반복 안내.
- 수익 서사(미검증): "500만 원 넘게 쓰며 얻은 노하우", "맨땅에 헤딩과 알고 시작하는 것의 차이", "앞으로 이 기능들로 수익 내는 방법을 하나씩 소개하겠다".
- CTA: 구독·좋아요·알림 설정을 누르면 "기발한 부업 방법"을 가장 빠르게 받아볼 수 있다며 마무리.

## 3줄 오리엔테이션
- Claude를 검색기처럼만 쓰지 말고 "제대로 세팅하면" 생산성과 수익을 올릴 수 있다는 전제로, 세팅 팁 9가지를 순서대로 따라하게 하는 홍보성 튜토리얼.
- 팁은 데이터 이관 → 모델 선택 → 아티팩트/인수인계 → 브라우저·데스크톱 연동 → 예약 자동화 → Claude Code → 디자인/피그마 순으로 난이도가 올라간다.
- 곳곳의 수익·절약 수치(500만 원, 70GB, 수백 시간)는 모두 화자 주장(미검증)이며, 무료 도구 배포·구독 유도 같은 세일즈 장치가 섞여 있다.

## 전문 — 시간순 워크스루

### [00:00–00:34] 도입 — "클로드를 고급 검색처럼만 쓰고 있다"
대부분의 사람이 Claude를 그냥 고급 구글 검색처럼 쓴다고 지적하며, 이를 "일론 머스크를 직원으로 고용해놓고 점심 메뉴나 고르게 하는 셈"이라고 비유한다.
> "마치 일론 머스크를 직원으로 고용한 다음에 점심 메뉴나 고르게 하고 있는 셈인 것입니다." [00:04]

화자는 자신이 지금까지 Claude에 500만 원이 넘게 사용해봤고, 실제로 Claude로 만든 앱으로 수익화에도 성공했다고 주장한다(미검증). 그래서 시간을 수백 시간 아껴주고 수익도 낼 수 있도록 생산성을 끌어올려줄 Claude 세팅법 9가지를 알려주겠다고 예고하고, 자신이 만든 생산성 도구도 무료로 줄 예정이니 끝까지 보며 따라하라고 안내한다(세일즈/유입 장치).

### [00:34–00:51] 팁1 — 다른 AI 데이터를 Claude로 통합
첫 번째 팁은 ChatGPT·Gemini에서 쓰던 데이터를 Claude로 통합하는 것. 이 기능을 쓰면 다른 AI에서 사용하던 설정값·대화 기록 등을 Claude로 옮겨준다. 특히 ChatGPT를 많이 쓸 텐데 2분 안에 옮기는 방법을 알려주겠다고 한다.

### [00:51–01:23] 팁1 실전 절차 — 가져오기 시작
Claude AI에 접속한 다음 좌측 하단의 내 아이디를 클릭하면 창이 뜨고, 거기서 설정으로 들어간 뒤 "기능" 버튼을 선택한다. 뜨는 기능들 중 "가져오기 시작" 버튼을 클릭하면 프롬프트 하나를 제공하는데, 그것을 복사해 쓰던 다른 AI에 붙여넣으면 내 정보 요약을 제공한다. 그 요약을 그대로 복사해 다시 Claude 창으로 돌아와 해당 부분에 붙여넣고 "메모리에 추가"하면 완료된다.

### [01:23–02:11] 팁2 — 모델별 특징과 토큰·사용 제한
두 번째로 Claude에는 다양한 모델이 있고 모델마다 쓰이는 토큰 양이 다르다. 용도별 추천[의견]: 잡담·시간 때우기는 하이쿠, 검색·간단한 서류 작업은 소넷, 회사 업무·숙제·앱/서비스 기획은 오퍼스. 단 오퍼스는 프로 구독을 해도 중간중간 사용이 제한될 수 있다.

제한을 설명하며 비유를 든다 — 하루 사용량이 1000이라면 100을 쓰고 1~3시간 정도 쉬어야 다음 100을 쓸 수 있는 식.
> "물론 이건 정확한 숫자가 아니라 이해를 돕기 위한 비유입니다." [01:59]

모든 대화에 오퍼스를 쓰면 유료 구독을 하고도 제대로 활용하기 어렵다며, 자신도 이 때문에 "수십만 원을 날린 적이 있다"고 주장(미검증)하고 주의를 당부한다.

### [02:11–02:59] 팁3 — 아티팩트로 자주 하는 작업을 도구화
세 번째로, 아무리 세팅을 잘해도 새 대화창에 들어가면 기존 대화 방식과 느낌이 다를 수 있으니, 자주 하는 작업은 미리 생산성 도구로 만들어 두면 좋다. Claude 사이트 좌측의 "아티팩트" 버튼을 클릭하면 도구를 빠르게 만들 수 있다. 화자는 영수증 관리, QR코드 만들기, 웹사이트 즐겨찾기 정리 도구를 만들어 쓰고 있으며, 이 도구들의 코드를 링크로 무료 제공하겠다고 안내한다(세일즈/유입 장치). 사용법 — 받은 코드를 다운로드한 뒤 새 아티팩트 → "처음부터 시작하기" → 다운로드한 코드 파일을 드래그해 넣고 "이대로 도구 제작해줘"라고 하면 끝. 필요한 도구를 요청해 자주 하는 작업을 편하게 관리하라고 권한다.

### [02:59–03:42] 팁4 — 인수인계 자료와 프로젝트로 대화 이어가기
네 번째로 Claude를 쓰며 불편했던 점 하나: 한 채팅방에서 너무 많은 작업을 하면 사이트가 느려지고, 이미지를 100개 넘게 넣으면 그 이후로는 이미지 업로드가 안 된다. 아무리 높은 요금제를 써도 마찬가지라고 주장한다. 대비책으로, 채팅 시작 시 "작업이 끝나면 인수인계 자료를 만들어줘"라고 미리 말해두면 작업 종료 후 인수인계 자료를 따로 만들어주고, 새 채팅방에 그 자료를 넣기만 하면 같은 환경으로 대화를 이어갈 수 있다. 또 좌측 프로젝트 버튼으로 같은 작업을 한 채팅방들을 모아두면 편리하다. 단 프로젝트에 넣는다고 자동으로 인수인계가 되는 것은 아니니 인수인계 자료는 꼭 만드는 습관을 들이라고 한다.

### [03:42–03:51] 팁4 보조 — 시스템 프롬프트로 통일
한 가지 팁을 더한다 — 시스템 프롬프트를 만들어 지침 박스에 붙여넣어 두면 통일된 방식으로 채팅방을 사용할 수 있다.

### [03:51–04:31] 팁5 — 크롬 확장 프로그램으로 브라우저 조작
다섯 번째로, Claude를 쓴다면 크롬 확장 프로그램을 꼭 설치하라고 권한다. 브라우저를 Claude로 조작하는 기능이다. 크롬 웹스토어에 접속해 클로드 관련 확장 프로그램을 검색해 설치하면, 브라우저에서 하는 대부분의 작업을 Claude가 비서처럼 도와준다. 예로 구글 스프레드시트 작업 시 엑셀 기능을 하나하나 검색할 필요 없이 도움을 받거나, 화면을 캡처하거나 검색할 단어가 생각 안 날 때 대충 적으면 그 단어를 유추해주기도 한다. 이 기능은 유료 플랜부터 사용 가능하다.
> [불명확: 원문 "클로드 폴롬을 검색하면" — 확장 프로그램명 음차로 추정, 정확 명칭 불확실]

### [04:31–05:08] 팁6 — 내 컴퓨터 안의 클로드(데스크톱)로 파일 정리
여섯 번째로 화자가 "진짜 미친 기능"[의견]이라며 강조하는 것은 내 컴퓨터 안에 Claude를 넣는 것(데스크톱 앱). 파일·폴더 정리가 매번 힘들었을 텐데, 별도 정리 프로그램을 깔면 악성 코드가 딸려오거나 비용이 많이 드는 반면 이를 Claude가 다 해준다는 논리. 구글에 클로드를 검색해 다운로드·설치하면 창이 뜨고, 화자는 미루던 파일 정리를 시켜봤더니 정리해줘서 "무려 70GB 정도를 확보했다"고 주장(미검증)한다.
> "정말 이 기능은 미친 기능이니까 꼭 써 보시기 바랍니다." [05:06]

### [05:08–05:41] 팁7 — 예약 작업(스케줄)으로 반복 자동화
일곱 번째로, 컴퓨터에 들어온 Claude를 비서처럼 활용할 차례. 매일 해야 할 반복 작업을 예약할 수 있는 기능이다. 방금 설치한 Claude PC에서 "스케줄"이라고 입력하면 입력한 작업을 예약 작업으로 만들어준다. "내 비서가 생긴 것"[의견]이라 생각하면 된다며, 컴퓨터가 켜져 있으면 매일 체크리스트에 맞게 작업을 예약하거나, 아침마다 뉴스를 요약해 보여주게 하거나, 바탕화면에 지저분하게 쌓인 작업 파일을 매일 정해진 시간에 특정 폴더로 정리하게 시킬 수도 있다고 예시를 든다.

### [05:41–06:16] 팁8 — Claude Code로 본격 서비스·수익화
여덟 번째로, Claude로도 간단한 도구는 만들 수 있지만 본격적인 코딩으로 서비스를 만들려면 Claude Code를 써야 한다. "개발자를 한 명 고용한다고 생각하면 된다"[의견]. 사용 방식은 두 가지 — 컴퓨터에서 앱처럼 쓰는 방법과 터미널을 통해 쓰는 방법. Claude에서 만드는 것이 생산성 도구 수준이었다면 Claude Code는 그걸 넘어 진짜 서비스를 만들 수 있는 도구이므로, 본격적 수익화를 위해 꼭 익혀두라고 강조한다. 이 Claude Code는 다른 영상에서 더 자세히 다루겠다고 예고한다.

### [06:16–06:51] 팁9 — 디자인 기능과 피그마 연동
아홉 번째로 Claude에 디자인 기능이 새로 나왔고 특히 웹 디자인에 매우 특화되어 있다. 주의점 하나 — Claude는 디자인 전문 툴에서 출발한 게 아니므로 앱 하나를 더 쓰면 "금상첨화"[의견]가 된다. 바로 피그마(Figma)다. 피그마로 초기 디자인을 잡고 이후부터 디테일을 Claude로 손보면 고퀄리티의 디자인을 만들 수 있다. 다만 Claude 디자인은 현재 유료 플랜 이상에서 미리보기 단계로만 사용 가능하므로, 피그마를 커넥터로 연결해 Claude와 함께 쓰는 것이 가장 베스트라고 권한다.

### [06:51–07:18] 마무리 — 수익 서사와 CTA
이렇게 Claude로 수익을 내기 위해 꼭 세팅해야 할 9가지를 알려줬다고 정리한다. 처음부터 다 설정할 필요는 없지만 알고 시작하는 것과 "맨땅에 헤딩"하는 것은 다르다며, 자신이 "500만 원 넘게 쓰며 알게 된 노하우"(미검증)를 담았고 앞으로 이 기능들로 수익을 만드는 방법도 하나씩 소개하겠다고 한다. 마지막으로 구독·좋아요·알림 설정을 누르면 "기발한 부업 방법"을 가장 빠르게 받아볼 수 있다며 마무리한다(CTA).
