## 요약 (주제별)

### 이 영상의 정체 — 5개 부업 몰아보기(컴필레이션)
- 하나의 새 방법이 아니라, 혼잡스 채널이 과거 개별 영상으로 소개했던 **부업 5종을 한 편에 묶은 편집본**이다. 각 부업은 (1) 과장된 수익 훅 → (2) 실전 따라하기 절차 → (3) 혼잡스 사이트 가입·무료자료·멤버십/이벤트 유입, 이 3단 구조를 반복한다.
- 다섯 부업: ① 전자 일기 판매 ② A4 손 사진 스톡 이미지 판매 ③ 캘리그라피 간판 상호 제작 판매 ④ 데이터 라벨링(크라우드웍스) ⑤ 배달 스티커 제작 판매.

### 부업 ① 전자 일기 작성·판매
- **주장**: 매일 퇴근길 5분, 노션에 일기만 쓰면 월 30만 원 이상, 본업화도 가능(주장/미검증). 지식·자료조사 불필요가 기존 전자책과의 차별점이라고 함.
- **절차**: 노션 가입 → 페이지 추가 → 목표 설정(예: "부업으로 매달 자동화 수익 10만 원 만들기") → 토글 목록으로 1일차·2일차… 매일 그날 한 일 기록, 성과 나면 캡처 사진 삽입 → 목표 달성까지 반복 → PDF로 내보내기 → 재능마켓에 판매.
- 화자는 과거 "블로그 방문자 1만 만들기" 과정을 전자책으로 만들어 팔았다고 함(주장). 지금은 전자책 퀄리티가 높아져 **혼잡스 무료 템플릿**으로 꾸며야 잘 팔린다며 사이트 가입을 유도. 주제가 막막한 사람용으로 "일자별 미션 템플릿"도 제공.

### 부업 ② A4 손 사진 → 스톡 이미지 판매
- **주장**: 마우스 클릭 몇 번, 10~15분 업로드로 월 30만~100만 원, "자면서 수익", "초 블루 오션"(주장/미검증). "이 영상 이후 길에서 A4 용지 들고 다니는 사람 많이 볼 것"이라는 표현 사용.
- **절차**: A4 용지 위에 손을 올려 폰으로 촬영 → 어도비 배경제거(누끼) 웹으로 배경 분리 → PNG 다운 → PNG→JPG 변환 사이트로 변환 → 프리픽(Freepik) 가입(become contributor), SNS 등록 시 승인 가점 → 파일 업로드·타이틀 입력·저장. 프리픽은 기본 10개 파일 업로드 후 검수.
- **확장(무한 소재)**: A4 용지를 밖으로 들고 나가 돌·자연물, 다이소·마트의 물건·과일·야채 밑에 깔고 같은 방식으로 촬영 → 수백~수천 개 생성. 프리픽 외 셔터스톡 등에도 다중 등록해 수익 확대 가능(주장).

### 부업 ③ 캘리그라피 간판 상호 제작·판매
- **주장**: 15분 작업으로 월 100만~200만 원, 화자가 "지금도 매달 200만 원 이상 이 방법으로 번다"고 함(주장/미검증). 건당 최소 10만~30만 원에 판매 가능(주장).
- **절차**: 미리캔버스 → 1000×1000 → 텍스트/폰트(포천 막걸리체)로 상호(예 "포천 돼지갈비") + 도장 요소 + 붓 느낌 돼지 요소. 단, 알려진 폰트라 사장님들이 기피하고 붓글씨 느낌이 약한 한계 → **혼잡스에서 "글씨 파츠" 무료 다운** 받아 미리캔버스에 올려 조합하면 "작가가 붓으로 쓴" 느낌으로 제작.
- **판매**: 네이버 블로그(이전 영상의 블로그 디자인법 참고) → 매일 1개씩 캘리그라피 상호 포스팅으로 포트폴리오 → 톡톡 파트너 가입·포스팅 등록으로 주문 접수. 한 달 꾸준히 하면 주문 들어온다고 함. 맞춤 파츠 제작은 혼잡스 홈페이지 문의(사업 확장 원하는 소수에게 저렴 제작 예정).

### 부업 ④ 데이터 라벨링 (크라우드웍스)
- **주장**: 하루 한 시간, 매달 100만 원 이상, 화자의 핵심 수익 파이프라인(주장/미검증). AI 발달로 전망 밝고 대기업·정부기관도 하는 "안전한" 사업이라고 함.
- **개념**: "로봇이 아닙니다" 캡차처럼, AI가 사물을 더 잘 분별하도록 이미지에 라벨을 다는 작업. 업체가 대기업·기관 의뢰를 받아 프로젝트 개설 → 사용자가 골라 수행.
- **절차**: 크라우드웍스 가입 → 라벨링 작업 → 프로젝트 선택 → QR로 앱 설치 → 가이드대로 사진 촬영·업로드 → 검수 후 포인트(현금화) 지급. 예: 해산물 프로젝트 10개 모아 수산시장에서 촬영하면 4만 원까지 가능(주장). 배지를 많이 받을수록 더 많은 프로젝트 오픈. 내일배움카드(국비지원) 대상이면 관련 교육 무료 수강 → 배지 취득. 자격은 HRD넷에서 확인.
- **제휴 이벤트**: 크라우드웍스가 혼잡스 구독자에 100만 원 지원 + 12만 원 상당 데이터 라벨링 교육 무료권 제공. 참여법: 크라우드웍스 가입 → 혼잡스.com → 라벨링 이벤트 설문(크라우드웍스 아이디 기재). 1·10·30·50·100번째 작성자에게 교육권, 콘텐츠 추천 1등 30만/2등 15만/3등 5만 원, 선착순 50명 스타벅스 쿠폰(주장/이벤트 조건).

### 부업 ⑤ 배달 스티커 제작·판매
- **주장**: 하루 1시간으로 월 100만 원 이상, "확실히 돈 되는 부업", 몇 업체가 스티커 판매를 독점하다시피 하는 초 블루 오션(주장/미검증).
- **절차**: 혼잡스.com → 배달 스티커 부업 카드 → 무료 36종 스티커 파일 다운 → 캔바(Canva) 가입 → 65×65mm → 파일 업로드 → 원형 도구 + 파츠 + 무료 폰트로 디자인, 그리기로 색칠 → 원형 돼지 파일 지우고 PDF 다운. 스마트스토어에 등록해 판매, 주문 들어오면 상호명만 바꿔 판매.
- **제작·배송**: 판매되면 혼잡스 제휴 성수동 인쇄소의 "배달 스티커 제작하기" 상품에서 옵션 선택·주문 → 주문번호 + 캔바 PDF 업로드. 주문부터 배송까지 인쇄소가 진행해 이후 신경 쓸 것 없음(주장).

### 세일즈·유입 구조 (반복 패턴)
- 다섯 부업 모두 결론이 **혼잡스 사이트(honjobs.com) 가입 → 무료 템플릿/파츠/스티커 다운로드**로 수렴한다. 이메일 가입 유도가 매 부업마다 반복.
- **멤버십**: 월 약 12,000원. 자료 지속 업데이트, 블로그·스마트스토어 교육자료, 상세페이지 제작 자료, 파츠 무한 제공, 배달 스티커 추가 시안·상세페이지 샘플 등이 혜택으로 제시됨.
- **제휴 이벤트**: 크라우드웍스와 연계한 100만 원 지원금 + 12만 원 무료 교육권 이벤트가 데이터 라벨링 파트의 핵심 유입 장치.
- 상당수 세부 판매 절차(재능마켓 판매, 스마트스토어 운영, 블로그 디자인, 톡톡 파트너 가입)는 "추후/이전 영상 참고"로 넘겨 이 영상 자체로는 완결되지 않는다.

### 리스크·주의 (정리자 관찰)
- 모든 수익 수치는 화자 1인 주장이며 근거·평균·성공률 제시가 없다. "자면서 수익", "초 블루 오션", "누구나 쉽게"는 반복되는 마케팅 어구다.
- 스톡 이미지(프리픽 검수), 재능마켓·스마트스토어 판매는 공급 과잉·승인 거절·경쟁 등 화자가 언급하지 않은 변수가 존재한다. 라벨링 포인트·이벤트 조건은 제휴 업체 정책에 종속된다.

### [21:32–22:13] 혼잡스 사이트·멤버십·무료 파일
www.혼잡스.com 검색 → 가입 → 메인에서 "배달 스티커 부업" 카드 클릭 → 두 가지 파일 다운로드 방식 표시. 좋은 자료를 지속 제공하기 위한 **멤버십**을 준비 중이라며, 가입 시 자료 지속 업데이트, 블로그·스마트스토어 교육 자료, 상세페이지 제작 자료·파츠 자료 등을 무한 제공하는 월 약 12,000원 멤버십이라고 소개. 멤버십이 아니어도 무료 제공 파일로 도전 가능하며, 비회원은 "무료 36종 스티커 파일"을 선택해 다운로드.

### [22:13–23:20] 캔바로 스티커 디자인
네이버에 "캔바(Canva)" 검색·접속 → 가입 → 시작하기 → 디자인 만들기 → 맞춤형 크기 → 가로 65 × 세로 65mm → 새 디자인 만들기 → 좌측 업로드 → 파일 업로드로 혼잡스에서 받은 파일 모두 업로드 → 원형 도구 파일을 올리고, 다양한 파츠와 캔바 무료 폰트로 디자인 → 그리기 항목으로 색칠까지 자유롭게 → 다양한 스타일의 배달 스티커 디자인 완성.

### [23:20–24:02] PDF 저장·스마트스토어 판매
불러왔던 원형 돼지 파일은 지운 뒤 공유 → 다운로드 → PDF로 저장. 이 파일을 스마트스토어에서 판매하고, 주문이 들어오면 상호명 입력 부분만 바꿔 판매. 스마트스토어 운영 과정은 (별도) 영상을 참고하면 쉽게 가능. 멤버십 가입 시 스마트스토어 상세페이지 샘플, 배달 스티커 추가 파츠, 블로그 홍보 방법 등을 추가 제공한다며 멤버십 관심을 재차 요청.

### [24:02–24:44] 인쇄 발주·배송
판매되면 공장에 인쇄 발주. 혼잡스 제휴 성수동 인쇄소를 통해 간편 인쇄 가능 — 혼잡스 사이트 → "혼잡스 인쇄소" → "배달 스티커 제작하기" 상품 클릭 → 주문 들어온 상품 옵션 선택 → 주문 완료 후 다시 상품 접속 → 파일 업로드 버튼에 주문번호 입력 + 캔바 PDF 업로드로 완료. 주문부터 배송까지 혼잡스 제휴 인쇄소가 진행해 이후 신경 쓸 부분이 전혀 없다고 함.

### [24:44–25:09] 마무리·CTA
배달 스티커 부업을 쉽고 재미있게 진행할 수 있다며 확실히 돈이 되니 꼭 도전하라고 권유. 멤버십 가입 시 추가 시안 제공과 판매 지원을 예고하며 멤버십 관심을 재요청. 내용이 마음에 들었다면 구독·좋아요를 부탁하며 마무리.
