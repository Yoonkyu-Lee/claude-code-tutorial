# AI로 영상 제작, 이미지 만들기 쉽게 만들기

> with2511 (기술루트 알렉) · 2026-06-24 · 길이 약 10:00 · [영상 링크](https://youtu.be/ZmugrBuT4Yc)
> 정리 방식: 시간순 심층 워크스루 (paraphrase) + 주제별 상세 요약

## 요약 (주제별)

### 무엇을 하는 영상인가
- Claude(클로드)에 Higgsfield(자막에는 "익스필드/엑스필드/히그스 필드"로 오인식됨) MCP를 커넥터로 연결해, 클로드 채팅 안에서 바로 영상과 이미지를 생성하는 과정을 시연한다.
- MCP(Model Context Protocol)는 클로드를 외부 도구·서비스와 연결해 주는 프로토콜로 설명된다. 여기서는 클로드를 영상/이미지 생성 AI와 이어 주는 "다리" 역할.

### MCP 커넥터 연결 절차 (실전 셋업)
- 클로드에서 설정 → 커넥터 → "맞춤 설정(커스텀 커넥터)" 화면으로 이동.
- "플러스(+) 커스텀 커넥터 추가"를 누르고, 이름(Higgsfield)과 URL(`mcp.higgsfield` 계열 주소 — 자막상 "MCP점 히그스 필드")을 입력해 추가.
- 추가 후 "연결"을 누르면 Higgsfield 사이트로 이동 → 사이트 가입/로그인 → 액세스 권한 요청에 "Allow(얼로우)" 승인.
- 다시 클로드로 돌아오면 커넥터 목록에 Higgsfield가 추가·활성화된 상태가 되고, 이때부터 동영상/이미지 생성을 요청할 수 있다.

### 사용 워크플로 (클로드와 대화 → 생성)
- 먼저 클로드와 대화로 아이디어를 다듬는다. 예: "드라마틱한 결과를 볼 만한 예시를 들어줘 — 요즘 유행하는 고전 소설을 영상/이미지로 만드는 예시, 프롬프트로 제공해 줘."
- 클로드가 후보(레미제라블/장발장, 모비딕, 노인과 바다, 삼국지 등 고전 소설)와 상세 프롬프트를 만들어 주고, 그 프롬프트로 Higgsfield 생성을 요청한다.
- 생성 시 클로드가 도구 사용(모델 호출) 허용을 요청하고, 승인하면 Higgsfield에서 제너레이팅이 진행된다.

### 영상 생성 시연
- 첫 예시로 고전 소설 소재 영상을 생성 → "생성이 시작됐습니다" 안내 후 결과물이 나온다.
- 사용된 영상 생성 모델은 자막상 "시덴스 2.0" [불명확: 원문 "시덴스 2.0", Seedance 2.0로 추정].
- 결과에 대해 화자는 "엄청나다", "길진 않지만 영상이 이렇게 만들어질 수 있다"고 평가 [의견]. 더 길게 요청하면 더 긴 영상이 나올 것 같다고 언급.
- 두 번째로 레미제라블(장발장) 영상을 생성. 클로드가 만든 프롬프트에는 19세기 프랑스 농민 복장의 남자가 비 젖은 파리 자갈길을 도주하고 헌병들이 횃불로 추격하는 장면, 로우앵글 트래킹샷 카메라 기법, 램브란트풍 키아로스쿠로 조명, 8K 영화적 필름 그레인 등 상세 연출·스타일 지시가 포함됨. 결과가 요청대로 거의 나왔다고 평가 [의견].

### 이미지·썸네일 생성 시연
- 영상뿐 아니라 이미지(썸네일)도 같은 방식으로 생성 가능.
- 이 영상을 소개하는 유튜브 썸네일을 요청 → "AI가 영화를 만든다 / Higgsfield MCP / 클로드" 문구가 들어간 이미지 생성.
- 추가로 "1인 개발 시대, 바이브 코딩 시대" 문구를 크게 넣고 배경에 코드가 깔리는 느낌의 썸네일, "기술루트 알렉" 채널명 포함을 요청 → 요청대로 결과가 나옴. 화자는 바로 쓸 수 있게 잘 나왔다고 평가 [의견].

### 핵심 메시지·활용 제안
- MCP의 장점: 클로드에서 작업을 이어가면서 영상 생성·이미지 생성 등을 하나로 연결해 쓸 수 있다는 점.
- 영상/이미지 제작이 필요한 사람은 클로드에서 MCP로 연결해 바로바로 이용하는 것도 좋은 방법이라고 제안 [의견]. 시청자에게 직접 따라 해 보길 권함.

## 3줄 오리엔테이션
- 클로드에 Higgsfield MCP 커넥터를 붙이면, 채팅 안에서 바로 AI 영상·이미지를 만들 수 있다는 것을 셋업부터 결과까지 시연하는 영상.
- 실전 흐름은 "클로드와 대화로 프롬프트 다듬기 → 생성 요청 → Higgsfield가 영상/이미지 출력"이며, 고전 소설 영상·유튜브 썸네일을 예시로 보여준다.
- 자막에서 브랜드명이 "익스필드/엑스필드"로 오인식되지만 실제는 Higgsfield(히그스필드) MCP다.

## 전문 — 시간순 워크스루

### [00:00–00:32] 인트로 — Higgsfield MCP 소개
채널 기술루트 알렉이 "심박한(신박한) AI 도구 하나"로 Higgsfield MCP를 소개한다 [불명확: 자막에는 "익스필드 MCP"로 나오나 이후 URL이 "히그스 필드"라 Higgsfield로 교정]. MCP는 Model Context Protocol로, 클로드를 외부의 다른 것과 연결해 주는 것으로 이해하면 된다고 설명한다. 오늘 보여줄 예제는 클로드와 연결해 바로 영상이나 썸네일을 제작하는 것.

### [00:32–01:15] 예제 개요 — 클로드에 MCP 연결
클로드를 외부 도구와 연결하는 방법이 있고, 그것을 연결해 "이러한 영상을 만들어 줘"라고 하면 Higgsfield MCP를 통해 실제 영상/이미지 생성 AI가 동작한다고 설명. 간단한 예제로 Higgsfield MCP를 클로드에 연결하는 것이며, 시청자가 따라 할 수 있게 보여주겠다고 한다 [불명확: 원문 "MC.aiP를 필드.aiam aimcp를" — 서비스/URL 음차 오인식].

### [01:15–02:01] 커넥터 추가 절차
클로드에 들어가 설정 → 커넥터(맞춤 설정)로 이동. 커넥터 화면에서 "플러스 커스텀 커넥터 추가"를 누른다. 이름은 Higgsfield, URL은 "MCP점 히그스 필드" 계열 주소를 입력하고 추가한다. 그러면 커넥터 목록에 들어온다.

### [02:01–02:43] 사이트 가입·권한 승인
추가된 커넥터에서 "연결"을 누르면 Higgsfield 사이트로 이동한다. 이 사이트를 한 번 가입해 줘야 한다. 액세스 권한을 요청하면 "얼로우(Allow)"를 눌러 승인하면 되고, 그렇게 하면 커넥터 연결이 완료된다.

### [02:43–03:12] 클로드로 복귀 — 커넥터 활성화 확인
다시 클로드로 돌아오면 커넥터 목록에 Higgsfield가 추가되어 활성화된 것을 볼 수 있다. 이 상태에서 동영상 또는 이미지를 요청할 수 있고, 그러면 Higgsfield를 통해 결과물을 얻게 된다.

### [03:12–04:03] 프롬프트 준비 — 고전 소설 소재
먼저 클로드에 "Higgsfield를 통해 드라마틱한 결과를 볼 만한 예시를 들어줘. 요즘 유행하는 고전 소설을 영상으로 만들거나 이미지를 생성하는 예시면 좋겠어. 프롬프트로 제공해 줘"라고 요청한다. 화자는 고전 소설이 요새 영상으로 많이 나오며 인기를 끄는 걸 봤다고 언급 [의견]. 클로드가 레미제라블, 모비딕, 노인과 바다, 삼국지 등을 후보로 제시한다. 이렇게 먼저 클로드와 충분히 대화한 뒤 생성을 요청할 수 있다는 흐름을 강조.

### [04:03–04:54] 영상 생성 시작
생성 요청을 실행하자 클로드가 모델 사용 허용을 요청하고, 이를 허용하니 Higgsfield에서 제너레이팅이 시작된다. "생성이 시작됐습니다"라는 안내가 뜨고, 영상을 만드는 모델로 "시덴스 2.0"이 표시된다 [불명확: 원문 "시덴스 2.0", Seedance 2.0로 추정].

### [05:27–06:40] 두 번째 영상 — 레미제라블(장발장) 프롬프트
장발장(레미제라블)을 소재로 또 하나 생성한다. 클로드가 만든 프롬프트를 읽어 보면: 너덜너덜하고 찢어진 19세기 프랑스 농민 복장의 절박한 남자가 비에 젖은 파리 자갈길을 미친 듯이 달리고, 젖은 어둠 속 가스등이 깜빡이며 저 멀리 헌병들의 실루엣과 횃불이 추격해 온다. 안개, 빗물, 이끼 낀 돌담의 그림자 등 묘사와 함께, 카메라 기법은 남자 바로 뒤 지면 가까이에서 시작하는 로우앵글 트래킹샷으로 좁은 파리 골목 질주를 따라가다 점차 멀어지며 미로 같은 안개 도시를 담는다.

### [06:40–07:12] 분위기·스타일 지시
분위기는 절박함·생존·영화적 긴장감·서사적 역사극. 스타일은 살아 움직이는 어두운 유화, 램브란트의 키아로스쿠로 조명, 깊은 그림자, 차가운 젖은 돌 위에 비치는 가스등의 따스한 불빛, 풍부하고 짙은 색감, 8K 영화적 필름 그레인, 회화적 텍스처. 이렇게 카메라 기법과 스타일까지 상세히 설정해 요청한 결과라고 설명.

### [09:24–09:59] 마무리 — MCP의 의미와 제안
결국 MCP는 클로드에서 뭔가를 계속 하면서 영상도 만들고 이미지도 만드는 것들을 다 연결해서 쓸 수 있다는 장점이 있다고 정리. 영상/이미지 제작이 필요한 사람은 클로드에서 MCP로 연결해 바로바로 이용하는 것도 좋은 방법일 수 있으니 한번 해 보길 권한다 [의견]. 좋은 내용으로 다시 찾아오겠다는 인사로 마친다.
