## 요약 (주제별)

### 배경 — OpenClaw 열풍과 SkyClaw의 포지셔닝
- 화자는 젠슨 황이 GTC 2026에서 "오픈클로(OpenClaw)는 인류 역사상 가장 인기 있는 오픈 소스 프로젝트"라고 언급했다고 전한다 (화자 전언/미검증).
- 엔비디아가 오픈클로 기반 파생 제품(자막상 "로우")을 발표하며 프라이버시·보안 제어를 강화하고 "단 한 줄의 명령어로 설치" 가능하다고 밝혔다고 소개한다 (화자 전언/미검증).
- 이 흐름 속에서 "일반 사용자(테크 관심층이 아닌)가 오픈클로를 더 쉽고 안전·효율적으로 쓰는 법"에 대한 답으로 SkyClaw(스카이클로)를 제시한다.
- SkyClaw는 화자가 국내에 거의 최초로 소개했다는 Skywork(스카이워크) AI 플랫폼에서 출시한, 오픈클로 아키텍처 기반 **24시간 개인용 클라우드 AI 어시스턴트**다.

### SkyClaw의 핵심 구조 — 클라우드 VM 상시 가동
- 오픈클로가 주목받는 이유는 단순 챗봇이 아니라 **실제 작업을 수행하는 실행형 에이전트 구조**를 보여줬기 때문이라고 설명한다.
- 문제는 일반인이 오픈클로류 시스템을 직접 설치·세팅하고 CLI 기반으로 운영하기엔 번거롭다는 점. SkyClaw는 별도 디바이스(노트북·맥미니 등)를 사지 않고 이를 **클라우드 기반으로 가볍게** 가져온 형태다.
- 복잡한 배포 없이 바로 시작 가능하고, 내 컴퓨터를 계속 켜둘 필요가 없다(클라우드에서 이미 24시간 가동). 로컬이 아니라 클라우드 VM에서 작동해 부담이 적다.
- 통신 방식: Skywork뿐 아니라 텔레그램·왓츠앱 등 오픈클로가 지원하는 채널에서 명령을 내릴 수 있다.
- **GUI 포함 클라우드 호스트를 1인당 한 대씩 제공**한다고 함. 가상 데스크톱 안에서 동작을 시각적으로 확인하고 필요 시 실시간 직접 제어 가능. 모델은 Claude Opus 등 프리미엄 모델을 선택할 수 있다.
- 일반 챗봇과의 차별점: 한 번 맡겨두면 클라우드에서 계속 돌아가며 추적·정리·결과물 연결까지 해준다는 점.

### 데모 1 — AI 반도체/HBM 이슈 추적 브리핑 + 발표자료
- 프롬프트 예시: "AI 반도체·HBM 관련 이슈를 계속 추적하며 중요한 변화가 있으면 핵심만 정리한 브리핑 문서를 만들고, 마지막에 5장짜리 요약 발표 자료 초안도 준비해줘."
- Skywork PPT 스킬을 통해 "나노바나(Nano Banana)급" 고품질 PPT를 빠르게 생성 가능하다고 함 ([의견]/벤더 주장). 서치·슬랙 등 여러 기능이 스킬로 미리 구성돼 있고, 오픈클로에서처럼 원하는 스킬을 가져와 쓸 수 있다(일부는 차단됨).
- 결과물: 촬영일(3월 18일) 기준 DOCX·PPT 파일이 홈의 Skywork 워크스페이스에 생성되고 다운로드 링크 제공. 그래프·발표자료가 자동 생성되고 이미지도 자동 삽입.

### 데모 2 — 리서치 보고서(출처 추적) + 비교표 + 차트 아이디어
- Skywork는 **430개 이상의 전문 데이터베이스 인용을 지원**한다고 함(화자 이전 소개 반복/벤더 주장). 참고 문헌이 자동으로 붙는 문서 작업이라고 표현.
- **GAIA 벤치마크 점수 82.42%**를 냈다고 강조(자막 "82.42%가 42%가 나왔거든요" — 수치 82.42%로 정리, 화자 주장/미검증). GAIA는 한 줄 답 찾기가 아니라 검색·다단계 작업을 섞어 해결하는 에이전틱 성격이라 점수가 높을수록 좋다고 설명.
- "AI 반도체 시장 전망" 보고서를 생성하며 외부 검색·최신 수치 분석을 자동 수행. 목차·표지 생성, 도표/이미지/그래프 삽입(웹 이미지 자동 수집 또는 출처 기반 직접 작도), 핵심 기업 비교표(섹션 7), 발표용 차트 아이디어(매트릭스·버블·그룹 막대 차트 등) 제안까지 이어진다.

### PPT 기능과 가격 구조
- Skywork는 발표용 PPT 제작에 심혈을 기울여 홍보하는 편. **무료로 최대 10장까지** 만들 수 있다.
- **월 구독 19.99달러**면 무제한 이용 가능(화자 안내). 템플릿 다수 선택 가능, 크리에이티브 중심(나노바나 2 사용)과 분석·전문화 스타일 구분 생성.
- 온라인 수정·다운로드 후 편집 가능. 폰트 변경, 표·차트 자동 생성, 데이터 서식·필터링·이미지 삽입·차트 편집 지원. 다운로드 포맷: PPT·PDF·HTML·Google Slides.
- 문서를 스프레드시트로 활용하거나 발표용 재료로 바로 반영 가능하다고 함.

### 데모 3 — HBM4/HBM3/SOCAMM 기술 브리핑, 썸네일·인트로 영상
- HBM4·HBM3·SOCAMM(자막 "쏘캠/소켓") 메모리 특성을 분석하는 기술 브리핑 생성. 목차·대역폭·용량·상세 스펙 정리, 비교표·차트 아이디어까지 하나의 분석 패키지로 구성.
- 유튜브 썸네일 생성(레이어 3종), 영상용 커버 비주얼("묵직하고 분석적인 메탈 계열" 요청 반영), 썸네일 가이드(레이아웃·타이틀 크기·색상 가이드) 제공.
- **인트로 영상 생성**: HBM4·HBM3·SOCAMM 주제의 "하드코어 테크" 짧은 인트로 영상을 16:9, 약 15초, 음성 포함으로 자동 생성. 단일 클립이 아니라 전체 스토리를 구성하는 "제작 스튜디오" 느낌이라고 평가([의견]).

### 화자 총평과 세일즈 구조
- [의견] Skywork AI 2.0은 단순 툴을 넘어 점점 "디지털 파트너"처럼 활용할 수 있다고 느꼈다. 흩어진 자료를 모으는 것뿐 아니라 문서·표·PPT·이미지·영상 같은 실제 결과물로 바꾸는 과정까지 밀어준다.

- 오픈클로를 SkyClaw라는 이름으로 클라우드 기반 24시간 가상머신 1대 호스팅 개념으로 쓸 수 있다는 점을 재강조. 무료로 시작 가능(PPT 10장 무료), 해외 결제 시 무제한.

Skywork는 오픈클로 흐름을 이해하면 쉽게 보인다며, 오픈클로가 주목받는 이유는 단순 챗봇이 아니라 실제 작업을 수행하는 **실행형 에이전트 구조**를 보여줬기 때문이라고 설명. 다만 오픈플로우류 시스템은 바이럴을 타며 다양한 시스템·에코가 생겼음에도, 일반인이 직접 설치·세팅하고 CLI 기반으로 운영하기엔 번거롭다는 점을 짚는다.
