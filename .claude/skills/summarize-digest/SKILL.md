# summarize-digest (Layer 2 — 주제별 상세 요약)

완성된 **paraphrase**(Layer 1)를 소스로, **주제 기준** 요약을 만들어 **같은 파일 맨 위에 삽입**한다. paraphrase는 이미 dedup·교정·라벨링이 끝난 깨끗한 소스라, 여기서 뽑는 요약이 raw 자막에서 뽑는 것보다 정확·일관하다.

## 입력
- Layer 1 paraphrase 파일 경로 (같은 파일에 요약을 삽입한다)

## 워크플로
1. **paraphrase 통독**: 시간순 전문을 읽는다.
2. **주제 도출**: 영상 순서가 아니라 **내용 주제**로 묶는다 (예: 방법론 / 수익·근거 주장 / 실전 절차 / 리스크·주의 / 세일즈 구조 등 — 영상마다 다름).
3. **주제별 상세 요약 작성**: 각 주제 아래 핵심 정보를 **다 담는다**. 스킴용 한 줄이 아니라, 그 주제에 대해 영상이 말한 실질을 상세히. (분량은 "생각보다 상세하게".)
4. **정확성 유지**: paraphrase가 "화자 주장/미검증"·`[의견]`·`[불명확]`으로 둔 것은 요약에서도 그대로 유지. 단정 금지.
5. **파일 상단 삽입**: 아래 위치에 요약 블록을 넣는다.

## 삽입 결과 파일 구조

```markdown
# [영상 원제]

> 메타 …
> 정리 방식: 시간순 심층 워크스루 (paraphrase) + 주제별 상세 요약

## 요약 (주제별)
### 주제 1
- 그 주제의 핵심 정보를 상세히 (정보 누락 없이)
### 주제 2
- …

---

## 3줄 오리엔테이션        ← 이하 기존 paraphrase 전문 그대로
## 전문 — 시간순 워크스루
…
```

- 요약과 전문 사이는 `---`로 구분. 요약이 먼저 보이고, 아래에 paraphrase 전문.

## 규칙
- **주제 기준**(시간순 아님). 회의에서 논의한 "주제별 종합 재구성" 방식.
- **정보를 다 담되** 표현은 요약. "상세한 요약" — 스킴용 축약과 다름.
- paraphrase에 없는 사실을 새로 만들지 않는다 (paraphrase가 유일 소스). 자막 재확인이 필요하면 그건 Layer 1의 일.
- 제목·메타는 paraphrase 것을 그대로 둔다.

## Negative Space
- 자막을 새로 가져오지 않는다 (paraphrase가 소스).
- paraphrase 본문(전문)을 수정하지 않는다 — 위에 요약만 얹는다.
