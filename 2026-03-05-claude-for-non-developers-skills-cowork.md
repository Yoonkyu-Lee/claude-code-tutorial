## 핵심 개념

### 1. 화자가 Claude로 돌아온 이유 (Opus 4.6 업데이트 기준)
화자는 자신을 Claude 팬으로 소개하면서, 작년에 잠깐 Gemini로 갈아탔다가 다시 돌아왔다고 밝힌다.

> "저는 클로드의 팬입니다. ChatGPT 나왔을 때도 Gemini 나왔을 때도 저는 클로드를 주력으로 써 왔거든요." [00:00]

> "솔직히 말씀드리면 작년에 잠깐 흔들렸어요. Gemini가 무료에 성능에 또 올라오면서 서브로 쓰다 보니 어느 순간 클로드 쓰는 빈도가 좀 줄더라고요. 근데 4.6 버전 업데이트 이후로 다시 확 돌아왔어요." [00:09]

[의견] 화자는 글쓰기·기획·업무 자동화 영역에서 Claude가 다른 AI와 비교가 안 될 정도라고 주장한다.

> "글 쓰기, 기획, 업무 자동화 이쪽에서 클로드가 다른 AI들이랑 솔직히 비교가 안 될 정도예요." [00:24]

화자가 Opus 4.6 업데이트 후 느낀 변화 세 가지:

1. **글쓰기 퀄리티 향상** [00:43] — 대본·기획 문서 작업. [의견] "대본, 강의 자료, 기획서처럼 글 쓰기가 핵심인 자료라면 지금 AI 중에 Claude가 가장 낫다고 생각해요." [00:57]
2. **Skill과 Cowork 기능 추가** [01:03] — 업무 방식을 저장해 두고 내 PC 파일을 직접 정리해 주는 기능.
3. **Claude Code의 코딩 성능** [01:11] — [의견] "코딩 실력 평가에서 현재 세계 1[위]이라고 하는데" — 화자는 이를 비개발자 시각에서 소개하겠다고 함. (벤치마크 출처는 영상에 미제시 — [불명확])

### 2. 모델 3종 선택 기준 (화자 기준)
Claude의 세 모델을 용도별로 구분한다.

| 모델 | 화자가 제시한 용도 | 특징 (화자 발언) |
|---|---|---|
| **Opus** (4.6) | 계약서 검토, 긴 보고서 분석, 여러 단계 기획서 작성 | 답변이 느리고 비용↑, 결과물 확실히 좋음 |
| **Sonnet** | 메일 초안, SNS 문구, 이메일, 간단한 분석 등 일상 업무 | Opus보다 훨씬 빠르고 비용↓ |
| **Haiku** | 에이전트 개발자용 | 일반 사용자에게는 비추천 |

> "Opus는 최고 성능 모델이에요. 계약서 검토, 긴 [보고서] 분석, 여러 단계를 걸쳐야 하는 기획서 작성 이런 데 써요. 답변이 좀 [느리]고 비용도 더 드는데 결과물이 확실히 좋아요." [01:31]

> "웬만한 글쓰기는 Sonnet만으로 충분해요. Haiku는 이거는 에이전트 개발자 용이기 때문에 [일반 사용자가] 쓰는 걸 추천드리지 않습니다." [01:48]

### 3. Skill — Claude에 전문 지식 심기 (영상의 핵심)
화자가 "오늘 영상에서 가장 중요한 부분"이라고 명시한 지점. [01:54]

> "요즘 AI 잘 쓰는 분들 보면 공통점이 하나 있어요. 매번 긴 설명 안 하는데 결과물이 딱 자기 스타일로 나오는 거죠. 비결이 바로 Claude Skill 기능이에요." [01:56]

Skill의 정의 (화자):
> "Skill을 한 줄로 설명하면 Claude한테 [특정] 분야 전문 지식을 심어 주는 기능이거든요." [02:05]

작동 비유: 카드 뉴스 Skill을 만들면 Claude가 카드 뉴스 전문 에디터처럼, 뉴스레터 Skill을 넣으면 뉴스레터 전문가처럼, 영업 제안서 Skill을 넣으면 영업 전문가처럼 행동한다. [02:08]

> "그래서 '카드 뉴스 만들어 줘' 한 마디만 해도 그 분야 전문가 수준으로 [뚝딱] 나오는 거예요. 아무 설명 안 했는데 대상, [문장]이 전부 딱 맞게 나오는 거 보고 소름 돋았거든요." [02:17]

### 4. Cowork / Projects — 내 작업 공간
화자는 Cowork를 "업무 방식을 저장해 두고 내 PC 파일을 직접 정리해 주는 기능"으로 소개 [01:04]. 정리 단계의 "둘째"에서는 **Projects 기능으로 전용 작업 공간**을 만들라고 권한다.

> "프로젝트 기능으로 내 전용 작업 공간부터 만들어 주세요. 목적별로 나누면 매번 설명 안 해도 내 스타일 그대로 나와요." [05:00]

### 5. Claude Code 맛보기 (비개발자 시각)
> "코딩을 전혀 몰라도 말로 시키면 Claude가 코드를 직접 짜 줍니다. 앱이나 웹사이트 만드는 거 코딩 하나도 모르는 사람도 할 수 있어요." [03:54]

- 브라우저에서 바로 써 볼 수 있는 **맛보기 버전**이 있다고 언급 [04:02] (이름·URL 미제시 — [불명확]).
- 실제 Claude Code는 **터미널 기반**: "컴퓨터에 있는 명령어 입력창" 비유. 내 컴퓨터 파일 전체를 읽고 여러 파일을 동시에 수정·실행. [04:09]

> "macOS에 최적화되어 있어 개발자들이 맥 쓰는 이유 중 하나가 이거라고 합니다. 저도 이것 때문에 맥북을 하나 더 구매할까도 고민 중이에요." [04:17]
> ([의견] macOS 최적화 주장은 화자가 전언("~라고 합니다") 형태로 제시 — 근거 미제시.)

## 팁 (실습 단계)

### Skill 만드는 법 (영상 실연)
1. 왼쪽 아래 클릭 → **설정** 탭 → **기능** 탭 → **Skill** 항목 [02:25]
2. **추가** 버튼. 처음이면 **대화로 만들기** 추천 — Claude가 "어떤 Skill 만들고 싶으세요?" 물으면 원하는 걸 말한다. [02:32]
   - [의견] "심지어 이전에 제가 대화했던 내용들 보고 제가 만들 것 같은 Skill을 추천까지 해 주더라고요." [02:40]
3. 만들어진 Skill을 **내 Skill에 복사**하면 바로 사용 가능 [02:44]

### Skill 직접 작성 시
- **Skill 이름은 영문으로** 써야 함 (예: `카드 뉴스` → 영문명) [03:02]
- 지침란을 모르겠으면 Claude에게 물어라: "카드 뉴스 제작할 때 좋은 지침 만들어 줘" → 답을 복사해 붙여넣기 [03:07]
- 결과물 수정도 대화로: "이 부분 좀 더 간결하게 바꿔줘" [03:34]

### 사용량(토큰) 주의 — 화자가 밝힌 단점
> "Claude는 token을 많이 써요. token이 뭐냐면요. Claude가 일을 할수록 소모되는 사용량이에요. 데이터 요금제처럼 쓰면 쓸수록 [닳는] 거라고 보시면 돼요." [04:28]

- 월 $20 Pro에서 Cowork 파일 정리 / Opus 긴 문서 작업을 하면 한도가 생각보다 빨리 참 [04:36]
- 본격 업무용은 월 $100 / $200 **Max 플랜**을 쓰는 사람도 많음 [04:43]
- [의견] 시작은 Pro로 먼저 써 보고 부족하면 올리는 걸 추천 [04:49]

## 환경 셋업
영상은 Claude **웹/데스크톱 앱**의 GUI 설정만 다룬다 (설정 → 기능 → Skill). 별도 CLI 설치 절차는 없다. Claude Code는 "맛보기 버전(브라우저)"과 "터미널 기반 실제 버전" 두 가지로만 언급되며 설치 명령은 제시되지 않는다.

## 외부 보강
영상이 Skill/Cowork/Projects/요금을 비개발자용으로만 설명하므로, 공식·2차 자료로 정의를 보강한다 (영상 내용과 분리).

- **Skill**: 한 번 정의해 두면 Cowork가 매번 같은 방식으로 실행하는 재사용 워크플로. Team/Enterprise 플랜에서는 동료·조직과 공유 가능(공유본은 보기 전용, 활성화·사용은 되나 편집 불가). 출처: [Use Skills in Claude — Claude Help Center](https://support.claude.com/en/articles/12512180-use-skills-in-claude)
- **Project**: 특정 도메인의 파일·컨텍스트·메모리를 담는 영속 작업 공간. 열면 Claude가 이미 맥락(무엇을 만드는지, 선호 방식, 중요 파일)을 알고 시작. 출처: [Cowork — Claude by Anthropic](https://claude.com/product/cowork)
- **Cowork 요금**: 2026-01-16 기준 Claude **Pro($20/월)에 Cowork 포함** (별도 요금 없음, 사용 한도 낮음). Max는 $100/$200/월로 한도↑. 출처: [Is Claude Cowork Included in Claude Pro? (2026 Guide)](https://www.jamout.ai/blog/is-claude-cowork-included-in-my-claude-pro-plan-yes-and-here-s-exactly-what-you-get-for-20)

> 영상의 "Cowork = 내 PC 파일 정리" 설명과 외부 자료의 "Cowork = 지식 노동용 실행 환경, Skill을 실행"이 같은 기능을 가리킨다. 영상은 소비자 관점 한 단면만 보여준 것.
